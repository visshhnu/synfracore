# GATE ECE — Fundamentals

## What is GATE ECE?

GATE ECE (Electronics and Communication Engineering) is the GATE paper for ECE graduates. It qualifies candidates for M.Tech at IITs/NITs and PSU jobs (BSNL, DRDO, BARC, BEL, ECIL, etc.).

## Exam Pattern

| Detail | Info |
|---|---|
| **Duration** | 3 hours |
| **Total Marks** | 100 |
| **Questions** | 65 (10 GA + 55 Technical) |
| **Negative Marking** | -1/3 for 1-mark, -2/3 for 2-mark MCQ |
| **NAT Questions** | No negative marking for Numerical Answer Type |

## Mathematics Topics (Must Master)

- Linear Algebra: matrices, eigenvalues, systems of equations
- Calculus: maxima/minima, integration, partial derivatives
- Differential Equations: first-order, second-order
- Probability: random variables, Bayes theorem, distributions
- Complex Numbers: phasors, complex power
- Fourier Series and Transforms
- Laplace Transforms
- Z-Transforms (for Digital Signal Processing)
