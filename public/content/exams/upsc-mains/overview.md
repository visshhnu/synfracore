# UPSC Mains — Overview
UPSC Mains selects candidates for IAS/IPS/IFS roles.
## Merit Papers (1750 marks)
Essay (250) + GS1 (250) + GS2 (250) + GS3 (250) + GS4/Ethics (250) + Optional 1+2 (500)
Interview: 275 marks | Total: 2025 marks
## Answer Writing Strategy
Structure: Introduction → Multi-dimensional analysis → Way forward
Focus on current examples, balanced views, constructive recommendations
