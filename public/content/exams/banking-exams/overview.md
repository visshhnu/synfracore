## Quantitative Aptitude — Key Topics & Shortcuts

```
HIGH PRIORITY (appears every year):
- Number System
- Percentage
- Ratio & Proportion
- Profit & Loss
- Simple & Compound Interest
- Time, Speed & Distance
- Time & Work
- Data Interpretation (DI)

SHORTCUTS TO MEMORIZE:

1. Percentage shortcuts:
   x% of y = y% of x
   % increase = (new - old)/old × 100
   Successive % change: a% then b% = a + b + ab/100

2. Simple Interest: SI = PRT/100
   Compound Interest: CI = P(1 + R/100)ⁿ - P
   CI - SI for 2 years = P(R/100)²

3. Profit & Loss:
   % Profit = (CP - SP)/CP × 100
   SP = CP(100 + %P)/100
   If article sold at x% loss and y% gain, break-even % = xy/(x or y)

4. Time & Work:
   If A does work in 'a' days, in 1 day A does 1/a work
   A+B together: 1/a + 1/b = (a+b)/ab → Days = ab/(a+b)

5. Speed Distance Time:
   D = S × T
   Average speed = 2xy/(x+y) when equal distances at speeds x and y

6. Number Series patterns:
   × 2, × 3 patterns
   +1, +2, +4, +8 (doubles)
   Difference of difference patterns
   Perfect squares/cubes
```

## Reasoning Ability — Master These

```
Syllogism:
Statements: All A are B. Some B are C.
Conclusions: I. Some A are C.   → Doesn't follow
             II. Some C are B.  → Follows (Some B are C → Some C are B)

Rules:
Universal Positive: All A are B
Universal Negative: No A is B
Particular Positive: Some A are B
Particular Negative: Some A are not B

Puzzle & Seating Arrangement:
Step 1: Identify the "anchor" (definite position)
Step 2: Place relative positions from anchor
Step 3: Eliminate using constraints

Blood Relations:
Create a family tree diagram
P is the son of Q. Q is the father of R. R is the brother of S.
→ P, R, S are siblings. Q is their father.

Coding-Decoding:
Look for patterns:
+1, +2 shift in alphabetical order
Reverse coding
Mathematical patterns on ASCII values
```

## English Language — Strategy

```
Reading Comprehension (RC):
- Read passage actively: mark key points
- Question types: main idea, inference, vocabulary, tone
- Answer only from passage, not general knowledge

Common Grammar Rules:
Subject-Verb Agreement:
  Neither/Either + singular verb
  None of the + plural noun + singular verb
  Each/Every/Either + singular
  Both/Many/Several/Few + plural

Active-Passive:
  Active:  Subject + Verb + Object
  Passive: Object + be + Past Participle + by + Subject

Connectors frequently tested:
  Although/Though/Even though (contrast)
  Since/Because/As (reason)
  Unless/Until (condition)
  Despite/In spite of + noun/gerund (contrast)

Cloze Test strategy:
  Read full paragraph first
  Check grammatical category (noun/verb/adj/adv)
  Check context (positive/negative)
  Check collocations
```

## General Awareness — Banking & Economy

```
RBI (Reserve Bank of India):
  Governor: Shaktikanta Das (as of 2024)
  HQ: Mumbai
  Founded: 1935, Nationalised: 1949
  Functions: Monetary policy, currency issue, banker's bank

Key Rates (memorize current values + what they mean):
  Repo Rate: Rate at which RBI lends to banks
  Reverse Repo: Rate at which RBI borrows from banks
  CRR (Cash Reserve Ratio): % deposits banks must keep with RBI (no interest)
  SLR (Statutory Liquidity Ratio): % deposits kept in liquid assets
  Bank Rate: Rate for long-term lending

Types of Banks:
  Scheduled Commercial Banks: SBI, PNB, HDFC, ICICI
  Small Finance Banks: AU, Equitas, Jana
  Payment Banks: Paytm, Airtel (can't lend)
  Cooperative Banks: UCBs, StCBs
  RRBs: Regional Rural Banks (sponsored by scheduled banks)

Financial Terms:
  NPA: Non-Performing Asset (loan unpaid 90+ days)
  NBFC: Non-Banking Financial Company
  CIBIL Score: Credit score 300-900 (750+ is good)
  LAF: Liquidity Adjustment Facility
  MSME: Micro, Small & Medium Enterprises
  
Current Affairs for Banking:
  → Follow RBI monetary policy committee decisions
  → Budget highlights (Union Budget)
  → New bank schemes and financial inclusion programs
  → International banking: SWIFT, FATF, IMF, World Bank
```

## 3-Month Preparation Plan

```
Month 1: Foundation
Week 1-2: Quant basics — Number system, Percentage, Ratio
Week 3-4: Reasoning — Coding-Decoding, Blood Relations, Inequalities
Daily: 1 RC passage, 20 vocab words

Month 2: Core Practice
Week 5-6: Quant — DI, Simplification, Time-Work-Speed
Week 7-8: Reasoning — Puzzles, Syllogism, Direction Sense
Daily: Previous year questions (100/day minimum)
GA: Start daily newspaper reading (The Hindu/ET)

Month 3: Mocks & Revision
Week 9-10: Full mocks (2 per week), analyze each thoroughly
Week 11: Sectional tests on weak areas
Week 12: Light revision, static GK, important dates

Mock Test Analysis (most important):
✓ Time spent per question (>2 min = skip)
✓ Questions you attempted wrong (concept gaps)
✓ Questions you skipped (time management)
✓ Pattern of mistakes (silly errors vs concept errors)

Target Accuracy Rate:
Prelims: Attempt 75-80, Accuracy 85%+ = clear cutoff
Mains: Attempt 110-120 out of 155, Accuracy 80%+
```
