# JEE Advanced — IIT Admission Exam Guide

## What is JEE Advanced?

JEE Advanced is the second stage of the Joint Entrance Examination. It is the only route to the 23 Indian Institutes of Technology. Only the top 2.5 lakh JEE Main qualifiers are eligible.

## Exam Structure

- Paper 1 and Paper 2 — both mandatory, 3 hours each
- Subjects: Physics, Chemistry, Mathematics (equal weightage)
- Question types: Single correct MCQ, Multiple correct MCQ, Numerical Answer Type
- Negative marking varies by question type

## JEE Advanced vs JEE Main

| Aspect | JEE Main | JEE Advanced |
|--------|---------|-------------|
| Difficulty | Moderate | Very High |
| Question types | Mostly MCQ | MCQ + Integer + Paragraph |
| Purpose | NITs, IIITs, GFTIs | IITs only |
| Attempts | 3 per year | 2 consecutive attempts |

## Topic Weightage

### Physics
- Mechanics (rotational, SHM): 8-10 marks
- Electricity and Magnetism: 8-10 marks
- Optics: 6-8 marks
- Modern Physics: 5-6 marks

### Chemistry
- Organic Chemistry (reactions, mechanisms): 12-15 marks
- Physical Chemistry: 8-10 marks
- Inorganic Chemistry: 8-10 marks

### Mathematics
- Calculus (integration, differential equations): 10-12 marks
- Algebra: 8-10 marks
- Coordinate Geometry: 5-7 marks

## JOSAA Counselling

JOSAA (Joint Seat Allocation Authority) conducts seat allocation for all IITs, NITs, IIITs, and GFTIs.

**Process:**
1. Register at josaa.nic.in with your JEE roll number
2. Fill program choices in priority order (up to 200 programs)
3. Lock choices before each round deadline
4. Seat allotted — pay fee and report to institute or float for next round

**Strategy:** Fill all programs you would genuinely attend. Use previous year opening and closing ranks from josaa.nic.in to build your list.
