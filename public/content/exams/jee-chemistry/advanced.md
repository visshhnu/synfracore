## Organic Chemistry Reactions

```
IMPORTANT NAME REACTIONS:

ALDEHYDE/KETONE REACTIONS:
  Aldol condensation: two aldehydes/ketones ‚Üí ő≤-hydroxy aldehyde/ketone (base)
  Cannizzaro: aldehyde with no őĪ-H ‚Üí disproportionation (acid + alcohol)
  Clemmensen reduction: C=O ‚Üí CH‚āā (Zn-Hg/HCl) ‚ÄĒ acidic conditions
  Wolff-Kishner reduction: C=O ‚Üí CH‚āā (NH‚āāNH‚āā/KOH) ‚ÄĒ basic conditions

  Hell-Volhard-Zelinsky: őĪ-halogenation using P and X‚āā (red P + Br‚āā)
  Koch reaction: alkene + CO + H‚āāO ‚Üí carboxylic acid (H‚Āļ)
  Kolbe reaction: sodium carboxylate + electrolysis ‚Üí alkane (coupling)

AROMATIC REACTIONS:
  Electrophilic aromatic substitution (EAS) ‚ÄĒ need catalyst for most:
    Nitration: conc HNO‚āÉ + conc H‚āāSO‚āĄ ‚Üí NO‚āā group
    Halogenation: X‚āā + Lewis acid (FeBr‚āÉ, AlCl‚āÉ) ‚Üí X group

    Friedel-Crafts acylation: RCOCl + AlCl‚āÉ ‚Üí COR group (no rearrangement)
  
  Directing effects:
    o,p directors (activate): -OH, -OR, -NH‚āā, -NHR, alkyl groups
    m directors (deactivate): -NO‚āā, -CHO, -COOH, -CN, -SO‚āÉH

AMINES:
  Gabriel synthesis: phthalimide + RX ‚Üí primary amine (no 2¬į or 3¬į)
  Hofmann bromamide degradation: RCONH‚āā + Br‚āā + KOH ‚Üí RNH‚āā (one C lost)
  Sandmeyer reaction: ArN‚āā‚Āļ ‚Üí ArX (X = Cl, Br, CN) using CuX
  Balz-Schiemann: ArN‚āā‚ĀļBF‚āĄ‚ĀĽ ‚Üí ArF + BF‚āÉ + N‚āā

REDUCTION REAGENTS:
  LiAlH‚āĄ (LAH): reduces aldehydes, ketones, esters, acids, amides, nitro

  H‚āā/Pd or Pt: hydrogenation of C=C, C‚Č°C, also reduces nitro to amine
```

## Coordination Chemistry

```
IUPAC NAMING:
  Order: anionic ligands ‚Üí neutral ligands ‚Üí cation name
  Ligand names: Cl‚ĀĽ=chlorido, NO‚āā‚ĀĽ=nitrito-N, NH‚āÉ=ammine, H‚āāO=aqua
  
  [Co(NH‚āÉ)‚āĄCl‚āā]Cl: tetraammineichloridocobalt(III) chloride
  [Pt(en)‚āā]¬≤‚Āļ: bis(ethylenediamine)platinum(II)

ISOMERS:
  Structural: ionisation (AgNO‚āÉ test), linkage (ambidentate ligand)
            hydrate, coordination position
  Stereo: geometric (cis-trans, fac-mer) | optical (non-superimposable mirror images)
  
  Square planar: cis-trans possible (MA‚āāB‚āā, MABC‚āā)
  Octahedral: cis-trans, fac-mer, optical isomers

VALENCE BOND THEORY (VBT):
  Hybridisation determines geometry
  Inner orbital complex: uses (n-1)d orbitals, low spin, diamagnetic (if paired)
  Outer orbital complex: uses nd orbitals, high spin, paramagnetic (if unpaired)

CRYSTAL FIELD THEORY (CFT):
  Strong field ligands (CO, CN‚ĀĽ, NO‚āā‚ĀĽ): large őĒ ‚Üí low spin
  Weak field ligands (I‚ĀĽ, Br‚ĀĽ, Cl‚ĀĽ, F‚ĀĽ, OH‚ĀĽ): small őĒ ‚Üí high spin
  Spectrochemical series: I‚ĀĽ < Br‚ĀĽ < Cl‚ĀĽ < F‚ĀĽ < OH‚ĀĽ < H‚āāO < en < CN‚ĀĽ < CO
```
