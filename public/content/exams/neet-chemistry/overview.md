# NEET Chemistry Overview

## Importance in NEET

Chemistry in NEET carries 180 marks — equal to Physics. However, Chemistry is generally considered more scoring than Physics because:
- Inorganic Chemistry is almost entirely from NCERT (factual, memorise once)
- Organic reactions follow patterns — understand once, apply everywhere
- Physical Chemistry requires fewer lengthy calculations than JEE

## Subject Division Strategy

```
INORGANIC CHEMISTRY (~40% of NEET Chemistry):
  Highest weightage in NEET (unlike JEE where Physical/Organic dominate)
  Source: 95% directly from NCERT Class 11 and 12
  Topics: periodic table, s/p/d/f block, coordination compounds, metallurgy
  Strategy: read NCERT line-by-line, memorize reactions and properties
  Timeline: 2-3 revisions mandatory (facts fade without revision)

ORGANIC CHEMISTRY (~35%):
  Reactions and mechanisms — conceptual understanding over memorisation
  NCERT sufficient for basics | need additional reference for mechanisms
  Topics: hydrocarbons, functional groups, name reactions, biomolecules
  Strategy: understand mechanism → predict products → practice PYQs

PHYSICAL CHEMISTRY (~25%):
  Calculation-based but formulas are straightforward at NEET level
  Topics: mole concept, thermodynamics, equilibrium, kinetics, electrochemistry
  Strategy: master mole concept first (foundation of all physical chemistry)
  Formulas: limited set, practice with numbers until automatic
```

## Topic-wise Marks Distribution (Approximate)

```
s-block and p-block: 12-15 marks
Coordination compounds: 5-8 marks
Organic reactions + name reactions: 15-20 marks
Biomolecules + Polymers: 5-8 marks
Mole concept + Stoichiometry: 4-6 marks
Chemical equilibrium + Ionic equilibrium: 6-8 marks
Electrochemistry: 5-8 marks
Chemical kinetics + thermodynamics: 6-8 marks
Atomic structure + bonding: 6-8 marks
Environmental chemistry: 3-4 marks (easy marks from NCERT)
```

## Study Plan

```
CLASS 11 FIRST (foundation):
  Physical: Basic concepts → Atomic structure → Bonding → Thermodynamics → Equilibrium
  Inorganic: Periodic table → s-block → p-block Part 1 (Groups 13-14)
  Organic: Basic principles → Hydrocarbons → Environmental chemistry

CLASS 12 (advanced):
  Physical: Solutions → Electrochemistry → Chemical kinetics → Surface chemistry
  Inorganic: p-block Part 2 (Groups 15-18) → d/f block → Coordination → Metallurgy
  Organic: Haloalkanes → Alcohols → Aldehydes/Ketones → Acids → Amines → Biomolecules

NCERT READING: 2 full readings minimum | 3-4 for inorganic
PYQ PRACTICE: after each chapter completion, do last 10 years of questions on that topic
MOCK TESTS: full NEET mocks from Month 7 onwards (3-hour stamina training)
```
