# JEE Physics Fundamentals

## Building the Physics Foundation

```
CLASS 11 FOUNDATION TOPICS:

UNITS AND DIMENSIONS:
  SI base units: metre, kilogram, second, ampere, kelvin, mole, candela
  Dimensional analysis: check equations, convert units, derive relations
  Significant figures: rules for addition, multiplication

KINEMATICS:
  Distance vs displacement | Speed vs velocity | Average vs instantaneous
  Equations of motion (uniform acceleration):
    v = u + at
    s = ut + ½at²
    v² = u² + 2as
    s(nth second) = u + a(2n-1)/2
  
  Graphs: slope of s-t = velocity | slope of v-t = acceleration | area of v-t = displacement
  Projectile motion: horizontal (no acceleration) + vertical (g downward)
    Time of flight: T = 2u sinθ/g
    Maximum height: H = u²sin²θ/2g
    Range: R = u²sin2θ/g | Maximum range at θ = 45°

NEWTON'S LAWS:
  1st Law: inertia — object continues in state unless external force acts
  2nd Law: F = ma (force = mass × acceleration)
  3rd Law: every action has equal and opposite reaction (different bodies)
  
  Free body diagram: isolate body, draw all forces (weight, normal, friction, tension)
  Friction: static (max = μₛN) | kinetic (= μₖN) | μₛ > μₖ always
  
  CIRCULAR MOTION:
    Centripetal force (towards centre): F = mv²/r = mω²r
    This is NOT an extra force — it IS the net force acting towards centre
    Vertical circle (car on hill, ball on string): write F = mv²/r at each point
```

## Class 12 Foundation

```
ELECTROSTATICS BASICS:
  Coulomb's law: F = kq₁q₂/r² where k = 9×10⁹ N·m²/C²
  Electric field: E = F/q = kQ/r²
  Electric potential: V = kQ/r (work done per unit charge)
  Relation: E = -dV/dr (field is negative gradient of potential)
  
  Gauss's Law: Φ = Q_enclosed/ε₀
    For sphere: E = Q/(4πε₀r²) outside | E = 0 inside conductor
    For infinite line: E = λ/(2πε₀r)
    For infinite plane: E = σ/(2ε₀)

CURRENT ELECTRICITY BASICS:
  I = Q/t (current = charge per time)
  V = IR (Ohm's law) | R = ρL/A (resistivity formula)
  Series: R_total = R₁+R₂+R₃ | same current, voltages add
  Parallel: 1/R_total = 1/R₁+1/R₂ | same voltage, currents add
  Power: P = VI = I²R = V²/R
```
