## Quantitative Aptitude

**Q: A shopkeeper marks goods 25% above cost and gives 10% discount. Profit %?**
```
Let CP = 100
MP (Marked Price) = 100 + 25% = 125
SP (Selling Price) = 125 - 10% of 125 = 125 - 12.5 = 112.5
Profit = SP - CP = 112.5 - 100 = 12.5
Profit % = 12.5/100 × 100 = 12.5%

Formula shortcut: Profit% = (1+m/100)(1-d/100) - 1 × 100
= (1.25)(0.9) - 1 = 1.125 - 1 = 0.125 = 12.5%
```

**Q: Ratio of speeds A:B = 3:4. A gives B a head start of 20m. 100m race?**
```
When B runs 100m, A runs 100 × 3/4 = 75m
A is 25m behind when B finishes.
With 20m head start for B: B only needs to run 80m
When B runs 80m: A runs 80 × 3/4 = 60m
B finishes at 80m | A is at 60m (A loses by 20m from finish = needs 40m more)
Wait — A started at 0, B started at 20. When B at 100 (running 80m), A at 60.
B wins by 100-60 = 40m? No — both start at same position, B gets 20m head start.
A has 100m to run | B has only 80m to run
Time for B: 80/4 = 20 units | Time for A: 100/3 = 33.3 units
B finishes first. At that time A ran: 20 × 3 = 60m. B wins by 40m.
```

## General Intelligence

**Q: Analogy — Book : Library :: Painting : ?**
```
Book is stored/displayed in Library
Painting is stored/displayed in Gallery/Museum
Answer: Gallery or Museum

SSC analogy tips:
  Identify the exact relationship (part-whole, tool-material, worker-place)
  Apply SAME relationship to second pair
  Watch for traps (superficially similar, not relationally similar)
```

**Q: Missing number in series: 2, 6, 14, 30, ?**
```
Differences: 4, 8, 16, ?
Pattern: differences double each time → next difference = 32
Next term: 30 + 32 = 62

OR: 2×2-1=3? No. Try: 2×1+4=6? 6×2+2=14? 14×2+2=30? 30×2+2=62 ✓
```
