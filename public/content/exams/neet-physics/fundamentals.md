# NEET Physics — Fundamentals

## Mechanics — Key Formulas

```
KINEMATICS (constant acceleration):
  v = u + at
  s = ut + ½at²
  v² = u² + 2as
  s_nth = u + a(2n-1)/2    ← displacement in nth second

NEWTON'S LAWS:
  F = ma (2nd law)
  Impulse = FΔt = Δp = m(v-u)
  Friction: f = μN (μₛ > μₖ)
  Normal force on incline: N = mgcosθ
  Acceleration on incline: a = gsinθ (frictionless)

CIRCULAR MOTION:
  v = rω, a_c = v²/r = rω²
  Centripetal force: F = mv²/r (directed inward)
  Banking angle: tanθ = v²/rg

PROJECTILE:
  Range: R = u²sin2θ/g (max at θ=45°)
  Max height: H = u²sin²θ/2g
  Time of flight: T = 2usinθ/g

GRAVITATION:
  F = Gm₁m₂/r² (Newton's law)
  g = GM/R² = 9.8 m/s²
  Escape velocity: v_e = √(2gR) = 11.2 km/s (Earth)
  Orbital velocity: v_o = √(gR) = 7.9 km/s (near Earth)
  Time period: T = 2π√(r³/GM) (Kepler's 3rd)
```

## Electricity — Key Formulas

```
ELECTROSTATICS:
  Coulomb: F = kq₁q₂/r² (k = 9×10⁹ Nm²/C²)
  Electric field: E = F/q = kq/r²
  Potential: V = kq/r, V = W/q
  Potential energy: U = kq₁q₂/r
  Capacitance: C = Q/V = ε₀A/d (parallel plate)
  Energy stored: U = ½CV² = Q²/2C

CURRENT ELECTRICITY:
  V = IR (Ohm's law)
  Power: P = VI = I²R = V²/R
  Resistivity: R = ρL/A
  Series: R = R₁ + R₂ + R₃
  Parallel: 1/R = 1/R₁ + 1/R₂
  Kirchhoff's: ΣV = 0 (loop), ΣI = 0 (node)
  Wheatstone bridge: P/Q = R/S (balanced, no current through galvanometer)
```
