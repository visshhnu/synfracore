# NEET Physics — Intermediate

## Optics

```
RAY OPTICS:
  Mirror formula: 1/f = 1/v + 1/u  (all signs from pole)
  Magnification: m = -v/u = h'/h
  Sign convention: distances measured from mirror/lens
    Object always on left → u is negative
    Concave mirror/converging lens: f is positive (NEW convention for NEET)

  Refraction (Snell's law): n₁sinθ₁ = n₂sinθ₂
  Refractive index: n = c/v = sinθ₁/sinθ₂
  Critical angle: sinC = 1/n (for n₂ < n₁)
  Total Internal Reflection: θ > C (optical fibres, diamonds)

  Lens formula: 1/f = 1/v - 1/u
  Lens maker's equation: 1/f = (n-1)[1/R₁ - 1/R₂]
  Power of lens: P = 1/f (in diopters, f in metres)
  Combined lenses: P = P₁ + P₂ + P₃

WAVE OPTICS:
  Young's Double Slit: fringe width β = λD/d
  Constructive (bright): path diff = nλ
  Destructive (dark): path diff = (2n-1)λ/2
  Brewster's angle: tanθ_B = n₂/n₁ (reflected light fully polarized)
```

## Modern Physics

```
DUAL NATURE:
  Photoelectric effect: KE_max = hν - φ (work function φ = hν₀)
  Einstein's equation: E = hν = hc/λ
  de Broglie wavelength: λ = h/mv = h/p
  Heisenberg uncertainty: Δx·Δp ≥ h/4π

ATOMIC STRUCTURE (Bohr model):
  Energy levels: Eₙ = -13.6/n² eV (for hydrogen)
  Radius: rₙ = 0.529n² Å (Bohr radius a₀ = 0.529 Å)
  Frequency of emitted photon: hν = E₂ - E₁
  Series: Lyman (n→1, UV), Balmer (n→2, visible), Paschen (n→3, IR)

NUCLEAR PHYSICS:
  Mass number A = Z + N (protons + neutrons)
  Binding energy: BE = Δm × 931.5 MeV (Δm in amu)
  Radioactive decay: N = N₀e^(-λt)
  Half-life: t½ = 0.693/λ
  Activity: A = λN = A₀e^(-λt) (measured in Becquerel or Curie)
  1 Curie = 3.7 × 10¹⁰ disintegrations/sec
```
