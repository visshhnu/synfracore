# PostgreSQL — Certification Guide

## EDB PostgreSQL Professional Certification

| Detail | Info |
|---|---|
| **Provider** | EnterpriseDB |
| **Exam Code** | EDB-PGPRO |
| **Cost** | $195 |
| **Validity** | Lifetime |
| **Format** | 75 questions, 90 min, MCQ |

### Domain Breakdown

- **25%** — Database design and DDL
- **20%** — SQL and DML
- **20%** — Database administration
- **20%** — Performance tuning
- **15%** — Security and backup/recovery

## Exam Tips

- PostgreSQL documentation is excellent — read it
- Focus on explain plans and index optimization
- Practice with pgbench for performance testing
- Understand MVCC, WAL, and vacuum
