# Mathematics Cheatsheet

## Algebra
```
(a+b)² = a²+2ab+b²;  (a-b)² = a²-2ab+b²
(a+b)(a-b) = a²-b²
(a+b)³ = a³+3a²b+3ab²+b³
a³+b³ = (a+b)(a²-ab+b²)
a³-b³ = (a-b)(a²+ab+b²)
Quadratic: x = (-b±√(b²-4ac))/2a
Sum AP: n/2[2a+(n-1)d]
Sum GP: a(rⁿ-1)/(r-1)
Infinite GP: a/(1-r), |r|<1
nCr = n!/r!(n-r)!;  nCr + nC(r-1) = (n+1)Cr
Binomial: (a+b)^n = Σ nCr a^(n-r) b^r
```

## Matrix Operations
```
det([[a,b],[c,d]]) = ad - bc
A⁻¹ = (1/det(A)) × adj(A)
det(AB) = det(A)det(B)
(AB)^(-1) = B^(-1)A^(-1)
Eigenvalue: det(A-λI) = 0
```

## Trigonometry
```
sin²+cos²=1;  1+tan²=sec²;  1+cot²=csc²
sin(A+B) = sinA cosB + cosA sinB
cos(A+B) = cosA cosB - sinA sinB
sin2A = 2sinA cosA;  cos2A = cos²A-sin²A
Sine rule: a/sinA = b/sinB = c/sinC
Cosine rule: a² = b²+c²-2bc cosA
```
