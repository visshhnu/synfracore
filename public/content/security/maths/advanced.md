# Mathematics Advanced Topics

## Real Analysis Foundations

### Sequences and Series Convergence
- Monotone Convergence Theorem: bounded monotone sequence converges
- Bolzano-Weierstrass: every bounded sequence has a convergent subsequence
- Cauchy criterion: sequence converges iff it is Cauchy
- Series tests: ratio test, root test, comparison test, integral test, alternating series test

### Continuity and Differentiability
- Epsilon-delta definition: f continuous at c iff for every ε>0, ∃δ: |x-c|<δ → |f(x)-f(c)|<ε
- Uniform continuity: same δ works for all x (stronger than pointwise continuity)
- Intermediate Value Theorem: continuous function on [a,b] takes all values between f(a) and f(b)
- Mean Value Theorem: ∃c ∈ (a,b) where f'(c) = (f(b)-f(a))/(b-a)

### Riemann Integration
- Upper and lower Riemann sums
- Riemann integrable: sup of lower sums = inf of upper sums
- Fundamental Theorem of Calculus: ∫[a to b] f = F(b)-F(a) where F'=f

## Linear Algebra — Advanced

### Eigenvalues and Eigenvectors
- Av = λv; characteristic polynomial det(A-λI) = 0
- Diagonalization: A = PDP⁻¹ where D diagonal (eigenvalues) and P (eigenvectors as columns)
- Conditions for diagonalizability: n linearly independent eigenvectors
- Spectral theorem: symmetric matrices have real eigenvalues and orthogonal eigenvectors

### Vector Spaces
- Definition: closed under addition and scalar multiplication
- Basis: linearly independent spanning set
- Dimension: number of basis vectors
- Rank-Nullity theorem: rank(A) + nullity(A) = n (number of columns)

### Inner Product Spaces
- Inner product: ⟨u,v⟩ with linearity, conjugate symmetry, positive definiteness
- Gram-Schmidt: orthogonalize a basis
- Projection: proj_v u = (⟨u,v⟩/⟨v,v⟩) × v
- Cauchy-Schwarz: |⟨u,v⟩| ≤ ||u|| × ||v||

## Abstract Algebra

### Group Theory
- Group (G,*): closure, associativity, identity, inverse
- Cyclic group: generated by single element
- Lagrange's theorem: order of subgroup divides order of group
- Homomorphism: φ(ab) = φ(a)φ(b); kernel = {g: φ(g) = e}
- Quotient group: G/N where N is normal subgroup

### Ring Theory
- Ring (R,+,×): abelian group under +, monoid under ×, distributive
- Integral domain: commutative ring with no zero divisors
- Field: integral domain where every nonzero element has multiplicative inverse
- Examples: ℤ (integers, ring), ℚ (rationals, field), ℤ_n (integers mod n, field iff n prime)

## Topology Basics

### Metric Spaces
- d(x,y) ≥ 0; d(x,y)=0 iff x=y; d(x,y)=d(y,x); triangle inequality d(x,z) ≤ d(x,y)+d(y,z)
- Open ball: B(x,r) = {y: d(x,y) < r}
- Open set: every point has a surrounding open ball within the set
- Closed set: complement of open set; contains all its limit points

### Compactness and Connectedness
- Compact: every open cover has a finite subcover; in ℝⁿ: closed and bounded (Heine-Borel)
- Connected: cannot be split into two disjoint open sets
- Path-connected: any two points can be connected by a continuous path (implies connected)

## Numerical Methods

### Root Finding
- Bisection: halve interval, converges O(1/2^n) — slow but guaranteed
- Newton-Raphson: x_{n+1} = x_n - f(x_n)/f'(x_n); quadratic convergence near root
- Secant method: similar to Newton but uses finite difference instead of derivative

### Numerical Integration
- Trapezoidal rule: O(h²) error; uses trapezoids under curve
- Simpson's rule: O(h⁴) error; uses parabolas; more accurate than trapezoidal
- Gaussian quadrature: optimal node placement for exact polynomials up to degree 2n-1
