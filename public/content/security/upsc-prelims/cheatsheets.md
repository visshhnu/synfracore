# UPSC Prelims Cheatsheet

## Quick Formulas

### Percentage
```
x% of y = xy/100
% increase = (increase/original) × 100
% decrease = (decrease/original) × 100
Successive %: A% then B% = A + B + AB/100
```

### Profit and Loss
```
SP = CP × (1 + P/100) or CP × (1 - L/100)
CP = SP × 100/(100+P) or SP × 100/(100-L)
Discount = Marked Price × Discount%/100
```

### Simple and Compound Interest
```
SI = PRT/100
CI = P(1 + R/100)^T - P
CI - SI (2 years) = P(R/100)²
CI - SI (3 years) = P(R/100)²(3 + R/100)
```

### Speed, Time, Distance
```
D = S × T; S = D/T; T = D/S
Average speed (equal dist) = 2ab/(a+b)
Train crossing pole: T = Length/Speed
Train crossing platform: T = (L_train + L_platform)/Speed
Relative speed (same dir): |S₁ - S₂|; (opposite dir): S₁ + S₂
```

### Ratio and Proportion
```
If a:b = c:d then ad = bc (cross multiply)
Compounded ratio: (a:b) and (c:d) = ac:bd
Mean proportional of a and b = √(ab)
```

### Time and Work
```
If A in 'a' days, rate = 1/a per day
Combined: 1/T = 1/a + 1/b + ...
If A and B together in T days, A alone = aT/(a-T)
```

## Reasoning Shortcuts
- **Alphabet series**: A=1, Z=26; count from both ends
- **Coding**: FACE in coding, find pattern of +/-/reverse
- **Blood relations**: draw family tree
- **Directions**: compass rose, final displacement
- **Syllogism**: Venn diagram method

## Important Numbers
- Days in each month (30 days has September...)
- Prime numbers up to 100
- Squares (1-25) and cubes (1-15)
- Important fractions: 1/7≈14.3%, 1/6≈16.7%, 1/8=12.5%
