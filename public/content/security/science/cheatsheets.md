# General Science Cheatsheet

## Physics Units
| Quantity | Symbol | SI Unit | Formula |
|---------|--------|---------|---------|
| Force | F | Newton (N) | F = ma |
| Energy/Work | E/W | Joule (J) | W = Fd cosθ |
| Power | P | Watt (W) | P = W/t |
| Pressure | P | Pascal (Pa) | P = F/A |
| Frequency | f | Hertz (Hz) | f = 1/T |
| Charge | Q | Coulomb (C) | Q = It |
| Voltage | V | Volt (V) | V = IR |
| Resistance | R | Ohm (Ω) | R = V/I |
| Current | I | Ampere (A) | I = Q/t |
| Capacitance | C | Farad (F) | C = Q/V |

## Key Chemistry
```
Ideal gas: PV = nRT
Equilibrium: Kc = [products]/[reactants]
ΔG = ΔH - TΔS (spontaneous if <0)
Rate = k[A]^m[B]^n
pH = -log[H⁺]; pOH = -log[OH⁻]; pH+pOH = 14
Nernst: E = E° - (0.0592/n) log Q
```

## Biology Quick Reference

### Cell Organelles
| Organelle | Function |
|-----------|---------|
| Nucleus | DNA storage, gene expression |
| Mitochondria | ATP production (respiration) |
| Chloroplast | Photosynthesis (plants) |
| Ribosome | Protein synthesis |
| Golgi apparatus | Sort, modify, ship proteins |
| Lysosome | Digest waste, foreign particles |
| ER (rough) | Protein synthesis + processing |
| ER (smooth) | Lipid synthesis, detox |

### Photosynthesis and Respiration
```
Photosynthesis: 6CO₂ + 6H₂O + light → C₆H₁₂O₆ + 6O₂
Respiration: C₆H₁₂O₆ + 6O₂ → 6CO₂ + 6H₂O + 36-38 ATP
```

### DNA/RNA Comparison
| Feature | DNA | RNA |
|---------|-----|-----|
| Sugar | Deoxyribose | Ribose |
| Strands | Double | Single |
| Bases | A, T, G, C | A, U, G, C |
| Location | Nucleus | Nucleus + cytoplasm |
| Function | Long-term storage | Protein synthesis |
