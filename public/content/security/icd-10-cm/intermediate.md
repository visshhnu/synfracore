# ICD-10-CM Intermediate

## Diagnosis Code Selection

### Specificity
Always code to the highest level of specificity documented:
- Use bilateral codes when both sides affected
- Specify laterality (right, left, bilateral)
- Include severity/stage when documented
- Include etiology when known

### Combination Codes
Single code that classifies:
- Two diagnoses
- Diagnosis with a secondary process (complication)
- Diagnosis with associated sign/symptom

Example: E11.65 — Type 2 diabetes mellitus with hyperglycemia
(Covers both the diabetes and hyperglycemia in one code)

## Diabetes Coding

### Type 1 vs Type 2
- Type 1: E10.x (immune-mediated, insulin-dependent)
- Type 2: E11.x (most common, initially managed with diet/oral meds)
- Secondary: E08.x, E09.x, E13.x (due to underlying condition or drug)

### Diabetes Complications
- With nephropathy: E11.21
- With retinopathy: E11.3x
- With peripheral neuropathy: E11.40
- With ketoacidosis: E11.10
- With foot ulcer: E11.621

## Neoplasm Coding

### Neoplasm Table
- Primary malignant: originating site
- Secondary malignant: metastatic (spread from primary)
- Carcinoma in situ: non-invasive
- Benign: non-cancerous
- Uncertain behavior: pathologist cannot determine benign vs malignant

### Sequencing Neoplasm Codes
- Active treatment: code neoplasm first
- Complication is reason for encounter: complication first
- Anemia due to neoplasm: code neoplasm first, then anemia

## Injury and Trauma Coding

### Fracture Codes
- Closed vs open fracture
- Displaced vs nondisplaced
- Laterality: right, left, unspecified
- Anatomic site: femoral head, shaft, condyle, etc.
- 7th character: A (initial), D (subsequent), G, K, P, S

### Burns
- Coded by degree (1st, 2nd, 3rd) and body site
- Total body surface area (TBSA) — T31 codes
- External cause codes for burns (fire, hot liquid, etc.)

## External Cause Codes

### Purpose
Describe how injury occurred, where, what activity, and status.
Not required but recommended for injury claims.

- V codes: Transport accidents
- W codes: Falls, drowning, fire, exposure
- X codes: Intentional self-harm, assault
- Y codes: Other external causes

### Place of Occurrence (Y93)
Y93 codes describe activity at time of injury.
