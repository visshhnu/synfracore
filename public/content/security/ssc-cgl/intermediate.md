## Advanced Quantitative Aptitude

### Number Theory
- **Euler's totient**: φ(n) = count of numbers ≤n coprime to n
- **Remainder theorems**: Fermat's little theorem, Wilson's theorem
- **Cyclicity of units digit**: powers of 2,3,7,8 repeat every 4
- **Divisibility tricks**: Sum of digits, alternating sum for 11

### Algebra — Advanced
```
(a+b)² = a² + 2ab + b²
(a-b)² = a² - 2ab + b²
(a+b)(a-b) = a² - b²
(a+b)³ = a³ + 3a²b + 3ab² + b³
a³ + b³ = (a+b)(a² - ab + b²)
a³ - b³ = (a-b)(a² + ab + b²)
```
