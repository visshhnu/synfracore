# CPT Fundamentals

## What is CPT?
Current Procedural Terminology — developed and maintained by the American Medical Association (AMA). Code system for reporting medical, surgical, and diagnostic procedures and services. Updated annually (effective January 1).

## CPT Code Types

| Type | Structure | Purpose |
|------|----------|---------|
| Category I | 5-digit numeric (00100-99999) | Mainstream procedures |
| Category II | 4 digits + F | Performance measurement tracking |
| Category III | 4 digits + T | Emerging technology (temporary) |

## Evaluation and Management (E/M) Codes

### Office Visit Types
- New patient (99202-99205): Has NOT received professional services from same physician/specialty group within past 3 years
- Established patient (99211-99215): HAS received services within past 3 years
- OB rule for new/established: If on-call or covering physician sees patient, use established code if regular physician is in same group

### 2021 E/M Guidelines
Level determined by EITHER:
1. **Medical Decision Making (MDM)**: Number/complexity of problems + data reviewed + risk of complications
2. **Total time** on date of encounter (includes non-face-to-face work that day)

### MDM Level Requirements
| Level | Code (New/Est) | Problems | Data | Risk |
|-------|---------------|---------|------|------|
| Straightforward | 99202/99212 | 1 self-limited | Minimal | OTC meds |
| Low | 99203/99213 | 2+ self-limited OR 1 stable chronic | Limited | Rx drugs |
| Moderate | 99204/99214 | 1 uncontrolled OR new problem | Moderate | Rx drug mgmt |
| High | 99205/99215 | 1+ threatening life/limb | Extensive | Hospitalization |

## Surgical CPT Basics

### Global Surgical Period
Time period post-surgery during which related care is included in the surgical fee:
- 000: No global period
- 010: 10-day global period
- 090: 90-day global period
- Pre-op services included: 1 day before for major, day of for minor
- Post-op services included: follow-up visits related to surgery

### Key Modifiers
| Modifier | Use |
|---------|-----|
| -25 | Significant, separately identifiable E/M same day as procedure |
| -26 | Professional component (interpretation only) |
| -50 | Bilateral procedure |
| -51 | Multiple procedures (reduced fee applies) |
| -59 | Distinct procedural service |
| -LT/-RT | Left/Right side |
| -TC | Technical component only |
