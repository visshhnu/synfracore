# CPT Advanced Topics

## NCCI Bundling Rules

### National Correct Coding Initiative
CMS edits identifying services that cannot be billed separately:
- Column 1 (comprehensive) and Column 2 (component): Column 2 is included in Column 1
- Modifier indicator 0: bundled, cannot override with modifier
- Modifier indicator 1: can override with appropriate modifier (e.g., -59)
- Modifier indicator 9: concept not applicable

### Common Bundling Scenarios
- Surgical approach (incision/closure) is always included in the surgical procedure
- Imaging guidance included in some procedures (check code descriptor)
- Local anesthesia included in most procedures
- Pre-op and post-op care included in global surgical package

## Anesthesia CPT

### Anesthesia Payment Formula
Total payment = (Base units + Time units) × Conversion factor
- Base units: assigned per procedure based on complexity/risk
- Time units: 1 unit per 15 minutes
- Physical status modifier adds units: P3 = +1, P4 = +2, P5 = +3

### Physical Status Modifiers (ASA Classification)
- P1: Normal healthy patient
- P2: Mild systemic disease
- P3: Severe systemic disease
- P4: Life-threatening systemic disease
- P5: Moribund patient
- P6: Brain-dead for organ donation

## Radiology — Professional vs Technical

### Component Billing
- Global code: both technical and professional components
- Modifier -TC: Technical component only (equipment, staff, film)
- Modifier -26: Professional component only (physician interpretation and report)
- When radiologist interprets at a hospital: -26 modifier; hospital bills -TC
- Private office with own equipment: bill global

## Emerging Technology (Category III)

### Category III to Category I Pathway
1. Technology emerges → Category III code assigned
2. Providers report using Category III → utilization data collected
3. After 5 years: either promoted to Category I (if evidence supports) or deleted
4. FDA clearance often required for Category I status

### Coverage Considerations
Category III codes NOT universally covered by payers.
Must check LCD (Local Coverage Determinations) and Medicare benefit policies.
Some require prior authorization.

## Compliance in CPT Billing

### Upcoding vs Downcoding
- Upcoding: billing higher-level code than documentation supports → overpayment → FCA violation
- Downcoding: billing lower-level code → leaving money on table → still poor coding
- Both are coding errors — accurate coding is the goal

### Documentation Audit Points
- E/M: MDM components or total time must be documented
- Surgical: operative report must support the procedure billed
- Radiology: interpretation report required for professional component billing
- Lab: ordering physician and medical necessity documentation
