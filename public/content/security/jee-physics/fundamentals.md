# JEE Physics Fundamentals

## Mechanics — Kinematics
- v = u + at; s = ut + ½at²; v² = u² + 2as (use g = 10 m/s² in JEE)
- Projectile: horizontal (ax=0, constant), vertical (ay=−g)
  - Range R = u²sin2θ/g; Max height H = u²sin²θ/2g; Time T = 2u sinθ/g
  - Maximum range at θ=45°: Rmax = u²/g

## Mechanics — Laws and Energy
- Newton's 2nd: F = ma (vector equation)
- Friction: static ≤ μsN; kinetic = μkN (μk < μs always)
- Centripetal: F = mv²/r = mω²r (directed inward toward centre)
- Work: W = F·d cosθ; Work-energy theorem: Wnet = ΔKE
- KE = ½mv²; PEgrav = mgh; PEspring = ½kx²
- Power: P = W/t = Fv

## Rotation
- τ = Iα; L = Iω; τ = dL/dt
- Parallel axis: I = Icm + Md²
- Rolling without slipping: vcm = ωR; total KE = ½mv² + ½Iω²

## Gravitation
- F = Gm₁m₂/r²; g = GME/RE²
- Escape velocity: ve = √(2GM/R) ≈ 11.2 km/s (Earth)
- Orbital velocity: vo = √(GM/r) = √(gR²/r)
- Kepler's 3rd law: T² ∝ r³

## Electrostatics and Circuits
- Coulomb: F = kq₁q₂/r² (k = 9×10⁹ Nm²/C²)
- Electric field: E = kq/r²; Potential: V = kq/r
- Capacitance: C = Q/V; parallel plate: C = ε₀A/d; Energy: U = ½CV²
- Ohm's law: V = IR; Power: P = VI = I²R = V²/R
- Kirchhoff: ΣI = 0 at junction (KCL); ΣV = 0 in loop (KVL)

## Magnetism and EM Induction
- Force on charge: F = qv×B; Force on wire: F = IL×B
- Faraday's law: EMF = −dΦ/dt (Φ = B·A)
- Lenz's law: induced current opposes change in flux

## Waves and Modern Physics
- Wave: v = fλ; Sound ≈ 340 m/s; Light = 3×10⁸ m/s
- Photoelectric: KE = hf − φ; stopping potential eV₀ = hf − φ
- de Broglie: λ = h/p = h/mv
- Bohr atom: En = −13.6/n² eV; rn = 0.529n² Å
- Radioactive decay: N = N₀e^(−λt); T½ = 0.693/λ; 1 u = 931.5 MeV/c²
