## Mechanics
```
v = u + at;  s = ut + ½at²;  v² = u² + 2as
F = ma;  f = μN;  W = Fd cosθ
KE = ½mv²;  PE_grav = mgh;  PE_spring = ½kx²
p = mv;  Impulse J = FΔt = Δp
τ = Iα;  L = Iω;  τ = dL/dt
F_gravity = Gm₁m₂/r²;  g = GM/R²
Orbital v = √(GM/r);  Escape v = √(2GM/R)
```

## Waves and Oscillations
```
SHM: x = A cos(ωt+φ);  T = 2π/ω
Spring: ω = √(k/m);  Pendulum: ω ≈ √(g/L)
Wave: v = fλ;  y = A sin(kx - ωt)
Standing wave: y = 2A cos(kx) sin(ωt)
Doppler: f' = f(v ± v_o)/(v ∓ v_s)
```

## Electricity and Magnetism
```
F = kq₁q₂/r²;  E = kq/r²;  V = kq/r
C = Q/V;  U = ½CV² = Q²/2C
V = IR;  P = VI = I²R = V²/R
F = qv×B;  F = IL×B (on wire)
Faraday: EMF = -dΦ/dt;  Lenz opposes change
Self-induction: EMF = -L dI/dt
LC: ω = 1/√(LC)
```

## Optics
```
Mirror: 1/v + 1/u = 1/f;  m = -v/u
Lens: 1/v - 1/u = 1/f;  m = v/u
Snell: n₁ sinθ₁ = n₂ sinθ₂
Critical: sin θc = n₂/n₁
Young's: β = λD/d
```

## Modern Physics
```
E_photon = hf = hc/λ
KE_max = hf - φ (photoelectric)
λ_dB = h/p = h/mv (de Broglie)
E_n = -13.6/n² eV (H atom)
r_n = 0.529n² Å
N = N₀ e^(-λt);  T½ = 0.693/λ
E = mc² (mass-energy)
```
