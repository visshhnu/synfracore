# Medical Coding Guidelines Advanced

## HCC (Hierarchical Condition Category) Coding

### Purpose and Impact
HCC codes used for risk adjustment in Medicare Advantage, ACOs, and MSSP programs.
Predict expected healthcare costs for the following year.
Higher-acuity diagnoses = higher HCC weight = higher capitation payment.

### HCC Coding Principles
- ALL chronic conditions must be coded EVERY YEAR (even if stable)
- Cannot rely on prior year's diagnoses — must be current encounter documented
- Specificity matters: CKD stage 3 vs CKD unspecified — different HCC values
- Both DM type and complications must be coded for full HCC capture

### High-Value HCC Conditions
- Diabetic complications (E11.21, E11.22, E11.40, E11.65)
- Heart failure (I50.22, I50.32)
- CKD stages 3-5 and ESRD (N18.3-N18.6)
- COPD (J44.1)
- Rheumatoid arthritis (M05.6-, M06.0-)
- Major depression (F32.2, F33.2)
- Dementia (F03.90)
- Peripheral neuropathy (G62.9)

## RADV (Risk Adjustment Data Validation)
CMS audits MA plans to validate HCC coding accuracy.
Invalid codes: no supporting documentation, not coded to highest specificity, no face-to-face encounter.
Penalties for overcoding: recoupment of capitation overpayments.

## Quality Measures and Coding

### HEDIS Coding Impact
- Many HEDIS measures rely on diagnosis coding
- Diabetes care: A1c testing, nephropathy screening, retinal exam
- Cardiovascular: statin use, blood pressure control
- Behavioral health: follow-up after hospitalization

### STARS Program (Medicare Advantage)
- CMS star ratings (1-5 stars) affect MA plan bonuses
- Better coding documentation = more accurate denominator identification
- Correct coding supports: identifying eligible patients, documenting care provided

## Fraud and Abuse in Coding

### Upcoding
- Reporting higher-level or more complex codes than documentation supports
- Example: billing 99215 when 99213 is supported
- Results in overpayment — must be returned; potential OIG investigation

### Unbundling
- Billing separately for components of a single procedure/service
- Example: billing CPT 99213 + 96372 when injection is the primary service with minimal E/M
- NCCI edits identify bundled code pairs

### Documentation Fraud
- Cloning or copy-paste notes without updating
- Backdating or falsifying records
- Signing charts for services not personally provided or supervised
- All constitute false claims under the False Claims Act

## Coding for New and Emerging Conditions
- ICD-10-CM updated annually October 1
- New codes often added for: COVID variants, new procedures, expanded specificity
- Always verify code validity for date of service — wrong year = claim rejection
- U codes (U07.1, U09.9): temporary emergency COVID codes now permanent
