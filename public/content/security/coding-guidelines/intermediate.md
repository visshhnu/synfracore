# Medical Coding Guidelines Intermediate

## Chapter-Specific Guidelines

### Chapter 1 — Infectious and Parasitic Diseases (A00-B99)
- **Sepsis**: Underlying organism code + R65.2x for severe sepsis
- **HIV**: B20 for confirmed HIV disease; Z21 for asymptomatic HIV status
- **COVID-19**: U07.1 confirmed; U09.9 post-COVID condition; Z20.822 exposure

### Chapter 2 — Neoplasms (C00-D49)
- Primary malignant: originating site — code first
- Secondary malignant (metastatic): secondary/spread
- In situ: non-invasive; Benign: non-cancerous
- Code neoplasm that's the reason for the encounter first
- Anemia due to neoplasm → neoplasm coded first, then anemia

### Chapter 9 — Circulatory System (I00-I99)
- **Hypertension**: Always presume cause-and-effect with heart disease and CKD when both present
- **Combination codes**: I11 (HTN+HF), I12 (HTN+CKD), I13 (HTN+HF+CKD)
- **Acute MI**: 7th character for initial encounter (A), subsequent (D), sequela (S)

### Chapter 10 — Respiratory (J00-J99)
- COPD with acute exacerbation: J44.1 (not separate codes)
- Pneumonia: code the specific type when known (organism identified)
- Asthma: specify severity (mild intermittent, mild persistent, moderate, severe) and status

### Chapter 13 — Musculoskeletal (M00-M99)
- Specify primary vs secondary, laterality, site
- Fractures: traumatic vs pathological vs stress; 7th character for encounter type

## Outpatient Coding Rules

### Key Differences from Inpatient
- Code to highest degree of certainty
- Do NOT code "probable," "suspected," "possible," "likely" diagnoses as confirmed
- Code the sign/symptom when diagnosis not confirmed
- Code chronic conditions as additional diagnoses when they affect care

### Encounter Types
- **New problem**: Code primary reason (sign/symptom if no diagnosis confirmed)
- **Established problem**: Code the condition, not just the check-up
- **Pre-op evaluation**: Code the condition requiring surgery as principal
- **Follow-up**: Code the appropriate follow-up code, resolved condition as history

## Inpatient Coding Rules

### Principal Diagnosis Selection
- Condition established AFTER STUDY to be chiefly responsible for the admission
- Not necessarily the admitting diagnosis
- "After study" = after all workup including labs, imaging, consults

### Uncertain Diagnoses (Inpatient Only)
- Code as confirmed: probable, suspected, likely, possible, consistent with, compatible with
- INPATIENT ONLY — this rule does NOT apply to outpatient settings

### POA (Present on Admission)
Required for inpatient claims:
- Y = Yes, present at admission
- N = No, developed during admission
- U = Unknown (documentation insufficient)
- W = Clinically undetermined

## Common Coding Errors to Avoid
1. Coding from index only without verifying in tabular
2. Coding probable/suspected diagnosis in outpatient
3. Coding symptoms integral to confirmed diagnosis
4. Not using combination codes when available
5. Missing required additional codes (organism, laterality)
6. Wrong sequencing (principal diagnosis selection error)
