# Medical Coding Guidelines Fundamentals

## What are Official Coding Guidelines?
Official rules for assigning ICD-10-CM codes. Published annually by CMS and NCHS. Supplement (not replace) coding conventions in the Tabular List and Index. Required for accurate coding and proper reimbursement.

## ICD-10-CM Structure

### Code Format
- 3-7 alphanumeric characters
- First character always a letter (A-Z, except U reserved)
- Decimal point after 3rd character
- Example: E11.621 = Type 2 DM with foot ulcer

### Alphabetic Index vs Tabular List
- **Step 1**: Look up condition in Alphabetic Index (get tentative code)
- **Step 2**: Verify in Tabular List (confirm specificity and conventions)
- NEVER code from the Index alone

## Key Coding Conventions

### Inclusion and Exclusion Notes
| Note | Meaning |
|------|---------|
| Includes | Conditions classified to this code |
| Excludes1 | Cannot code together — mutually exclusive |
| Excludes2 | Not included here — can code both if both present |
| Use additional code | Add second code for manifestation or detail |
| Code first | Sequence underlying condition first |
| Code also | Additional code may be needed |

### NEC and NOS
- **NEC** (Not Elsewhere Classified): Documentation is specific, but no specific code exists
- **NOS** (Not Otherwise Specified): Documentation is vague or unspecified
- NEC → use the specific code available even if not a perfect fit
- NOS → default/unspecified code; query provider for more specificity

## General Coding Principles

### Signs and Symptoms
- Code signs/symptoms when no definitive diagnosis established
- Do NOT code signs/symptoms integral to a confirmed diagnosis
- Exception: code additional signs/symptoms not routinely associated with the diagnosis

### Acute and Chronic
- If both acute and chronic versions of a condition exist as separate codes: code both (acute first)
- If same code covers both: use that code

### Sequencing Rules
| Setting | First Code | Term Used |
|---------|-----------|----------|
| Inpatient | Most resource-intensive condition after study | Principal diagnosis |
| Outpatient | Primary reason for encounter | First-listed diagnosis |

### Laterality
- ICD-10-CM has bilateral codes for many conditions
- Specify right, left, or bilateral when documented
- Unspecified laterality only when truly not documented

## Documentation Requirements
- Diagnoses must be documented by the treating/responsible provider
- Clinical Documentation Improvement (CDI): improve specificity through provider queries
- Non-physician providers (NPs, PAs) documentation acceptable per payer policy
- Lab/radiology reports alone cannot support code assignment without physician interpretation
