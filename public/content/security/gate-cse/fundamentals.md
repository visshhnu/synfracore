# GATE CSE Fundamentals

## About GATE CSE
GATE (Graduate Aptitude Test in Engineering) Computer Science and Engineering tests core CS fundamentals. Used for M.Tech admissions and PSU jobs.

## Exam Pattern
- 65 questions, 100 marks, 3 hours
- General Aptitude: 15 marks
- Technical: 85 marks
- Negative marking: 1/3 for MCQ

## Core Subjects

### Discrete Mathematics
- Set theory, relations, functions
- Propositional and predicate logic
- Graph theory: trees, paths, connectivity
- Combinatorics: permutations, combinations, pigeonhole principle
- Probability: Bayes theorem, conditional probability

### Data Structures
- Arrays, linked lists, stacks, queues
- Trees: BST, AVL, heaps, B-trees
- Graphs: BFS, DFS, shortest paths, MST
- Hashing: collision resolution
- Time and space complexity analysis

### Algorithms
- Divide and conquer: merge sort, quick sort, binary search
- Dynamic programming: LCS, knapsack, matrix chain multiplication
- Greedy: activity selection, Huffman coding
- Graph algorithms: Dijkstra, Bellman-Ford, Floyd-Warshall, Kruskal, Prim
- NP-completeness basics

### Theory of Computation
- Regular languages and finite automata (DFA, NFA)
- Context-free languages and pushdown automata
- Turing machines
- Decidability and undecidability
- Time and space complexity classes (P, NP, NP-complete)

### Computer Organization
- Number systems: binary, hex, 2's complement
- Boolean algebra and logic gates
- Combinational circuits: adders, multiplexers
- Sequential circuits: flip-flops, registers, counters
- Memory hierarchy: cache, main memory, virtual memory
- Pipelining: hazards, forwarding

### Operating Systems
- Process management: scheduling, synchronization, deadlocks
- Memory management: paging, segmentation, virtual memory
- File systems
- I/O management

### Databases
- Relational model, SQL
- Normalization (1NF-BCNF)
- Transaction management, ACID
- Indexing and query optimization

### Computer Networks
- OSI and TCP/IP models
- Data link: framing, error control, MAC
- Network layer: IP addressing, routing algorithms
- Transport layer: TCP vs UDP, flow control, congestion
- Application layer: HTTP, DNS, SMTP

### Programming Languages
- Scoping and binding
- Parameter passing mechanisms
- Compiler design: lexing, parsing, semantic analysis, code generation
- Runtime environments

## Study Strategy
1. NPTEL lectures for conceptual understanding
2. Previous year papers (2014-2024)
3. Made Easy / ACE Academy notes
4. Practice standard textbooks (CLRS for algorithms, Tanenbaum for OS/Networks)
5. Mock tests under exam conditions
