## Adult CPR Steps
```
1. Check scene safety
2. Tap shoulders: "Are you OK?" — no response
3. Call 112 / send someone for AED
4. Check breathing (no more than 10 seconds)
5. If not breathing normally — begin compressions:
   • Hand position: lower half of sternum, center of chest
   • Heel of dominant hand, other hand on top, fingers laced
   • Arms straight, shoulders directly over hands
   • Depth: ≥5 cm (2 inches) — compress firmly
   • Rate: 100-120/min (Stayin' Alive beat)
   • Allow full chest recoil between compressions
   • 30 compressions : 2 rescue breaths (if trained)
6. Attach AED as soon as available — follow voice prompts
7. Continue until: breathing returns / EMS arrives
```

## RICE for Musculoskeletal Injuries
```
R = Rest       Stop the activity immediately
I = Ice        20 min on / 20 min off (wrap ice — no direct contact)
C = Compression  Elastic bandage — firm but not cutting off circulation
E = Elevation  Raise injured limb above heart level
```

## Burns Protocol
| DO | DON'T |
|----|-------|
| Cool with running water 10-20 min | Use ice or ice water |
| Cover with non-adherent dressing | Apply butter, oil, toothpaste |
| Remove jewellery near burn | Burst blisters |
| Seek care for serious burns | Wrap tightly |

Seek immediate care if: >palm-size, face/hands/feet/genitals, 3rd degree, chemical/electrical, smoke inhalation.

## Choking — Age-Specific Response
| Age | Conscious | Unconscious |
|-----|-----------|------------|
| Adult / Child (>1 yr) | 5 back blows + 5 abdominal thrusts (Heimlich), alternate | Start CPR; look in mouth before each breath |
| Infant (<1 yr) | 5 back blows + 5 chest thrusts, alternate | Start CPR; look in mouth before each breath |
| Pregnant / Obese | Chest thrusts instead of abdominal thrusts | — |

## Recognition Quick Cards
**Anaphylaxis:** Allergen exposure + skin reaction (hives/swelling) + breathing difficulty → EpiPen outer thigh + call 112 immediately
**Stroke — FAST:** Face drooping + Arm weakness + Speech slurred + Time to call 112
**Heart attack:** Chest pressure + radiation to arm/jaw/back + sweating → 112 + Aspirin 300 mg (chew, if no allergy, not given to children)
**Seizure:** Protect from injury, don't restrain, time it, recovery position after → call 112 if >5 min or first episode
