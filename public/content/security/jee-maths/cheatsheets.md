# JEE Mathematics Cheatsheet

## Algebra Quick Reference
```
(a+b)² = a² + 2ab + b²
(a-b)² = a² - 2ab + b²
(a+b)(a-b) = a² - b²
(a+b)³ = a³ + 3a²b + 3ab² + b³
a³+b³ = (a+b)(a²-ab+b²)
a³-b³ = (a-b)(a²+ab+b²)
Quadratic: x = (-b ± √(b²-4ac)) / 2a
Sum of n terms AP: Sn = n/2 [2a + (n-1)d]
Sum of n terms GP: Sn = a(rⁿ-1)/(r-1)
Sum to infinity GP: S = a/(1-r), |r|<1
```

## Trigonometry
```
sin(A±B) = sinA cosB ± cosA sinB
cos(A±B) = cosA cosB ∓ sinA sinB
tan(A+B) = (tanA+tanB)/(1-tanA tanB)
sin2A = 2sinA cosA
cos2A = cos²A-sin²A = 1-2sin²A = 2cos²A-1
sin3A = 3sinA - 4sin³A
cos3A = 4cos³A - 3cosA
```

## Coordinate Geometry
```
Distance: d = √((x₂-x₁)²+(y₂-y₁)²)
Midpoint: ((x₁+x₂)/2, (y₁+y₂)/2)
Section m:n: ((mx₂+nx₁)/(m+n), (my₂+ny₁)/(m+n))
Area of triangle: ½|x₁(y₂-y₃)+x₂(y₃-y₁)+x₃(y₁-y₂)|
Slope m, intercept c: y=mx+c
Perpendicular slopes: m₁×m₂ = -1
```

## Conic Sections Quick Reference
| Conic | Equation | Key Feature |
|-------|---------|------------|
| Parabola | y²=4ax | Focus (a,0), directrix x=-a |
| Ellipse | x²/a²+y²/b²=1 | e=√(1-b²/a²)<1 |
| Hyperbola | x²/a²-y²/b²=1 | e=√(1+b²/a²)>1 |
| Circle | x²+y²=r² | e=0 |
