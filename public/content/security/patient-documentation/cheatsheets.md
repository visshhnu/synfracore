# Patient Documentation Cheatsheet

## SOAP Note Template
```
S (Subjective): Patient's reported symptoms, complaints, history
   HPI: OLDCARTS — Onset, Location, Duration, Character,
   Aggravating/Relieving factors, Timing, Severity
O (Objective): Measurable findings
   Vital signs: BP, HR, RR, Temp, SpO2, Weight
   Physical exam findings by system
   Lab/imaging results
A (Assessment): Clinical impression
   Problem list or diagnoses (most important first)
   Differential diagnoses if working up
P (Plan): What you will do
   Medications (name, dose, route, frequency, #days)
   Orders (labs, imaging, referrals)
   Patient education provided
   Follow-up timing
```

## Required Documentation Timing
| Document | Required Timing |
|---------|----------------|
| History and Physical | Within 24 hours of admission |
| Progress notes | Daily for inpatients |
| Operative report | Immediately post-surgery |
| Discharge summary | Within 30 days of discharge |
| Death summary | Within 24 hours |
| Procedure notes | Immediately after procedures |

## MDM Documentation Requirements
| Level | Problems | Data | Risk |
|-------|---------|------|------|
| Straightforward | 1 self-limited | Minimal | Minimal/OTC |
| Low | 2 self-limited OR 1 stable chronic | Lab/imaging ordered | Rx drug |
| Moderate | 1 uncontrolled OR new problem | Independent interpretation | Rx drug management |
| High | 1+ severe threat to life/limb | Extensive: external records + discussion | Hospitalization |

## Documentation Quality Indicators
✓ Signed and dated by responsible provider
✓ Legible (or dictated/electronic)
✓ Complete — all required elements present
✓ Accurate — consistent across the medical record
✓ Specific — uses definitive language, not vague
✓ Timely — completed within required timeframe
✓ Supports diagnoses coded on claim

## Common Documentation Deficiencies
- Missing or unsigned physician notes
- Incomplete diagnoses (lacking specificity)
- Late entries not labeled as such
- Inconsistency between different providers' notes
- Copy/paste without updating (cloning)
- Signs and symptoms not linked to diagnosis
