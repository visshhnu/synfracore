# Placement Preparation Fundamentals

## Overview
Campus placement requires preparation across: aptitude tests, coding rounds, technical interviews, and HR interviews.

## Aptitude Test Topics

### Quantitative Aptitude
- Number system: HCF, LCM, remainders, unit digits
- Percentages, profit/loss, simple/compound interest
- Time, speed, distance; time and work
- Ratio and proportion, mixtures and allegations
- Permutations, combinations, probability

### Logical Reasoning
- Number series, letter series, coding-decoding
- Blood relations, directions
- Syllogisms, statements and conclusions
- Data sufficiency

### Verbal Ability
- Reading comprehension
- Vocabulary: synonyms, antonyms, fill in the blanks
- Grammar: error identification, sentence correction
- Para jumbles

## Data Structures and Algorithms (DSA)

### Must-Know Data Structures
- Arrays, strings, linked lists
- Stacks, queues
- Trees (binary trees, BSTs, heaps)
- Graphs
- Hash maps/tables

### Must-Know Algorithms
- Sorting: bubble, selection, insertion, merge, quick
- Searching: linear, binary
- Graph: BFS, DFS, Dijkstra
- Dynamic programming basics
- Recursion and backtracking

## Core CS Subjects

### Operating Systems
- Processes, threads, scheduling
- Memory management: paging, virtual memory
- Deadlocks, synchronization

### DBMS
- SQL: SELECT, JOIN, GROUP BY, subqueries
- Normalization (1NF-3NF)
- Transactions, ACID properties
- Indexes

### Computer Networks
- OSI model, TCP/IP model
- TCP vs UDP, HTTP/HTTPS
- DNS, DHCP, routing basics

### OOP Concepts
- Abstraction, Encapsulation, Inheritance, Polymorphism
- SOLID principles
- Design patterns (Singleton, Factory, Observer)

## Resume Building
- One page for freshers
- Action verbs: developed, implemented, optimized, designed
- Quantify impact where possible
- Projects: problem statement, tech stack, your role, outcome
- Keep it clean, no typos, ATS-friendly format
