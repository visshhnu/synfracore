# HCPCS Intermediate

## DME Coding Deep Dive

### Coverage Requirements
Four criteria for Medicare DME coverage:
1. Medically necessary
2. Used in the home
3. Serves a medical purpose
4. Can withstand repeated use (durable)

### Certificate of Medical Necessity (CMN)
Required for:
- Oxygen (HCPCS E0424-E0452)
- Power wheelchairs (K0800-K0899)
- Hospital beds (E0250-E0316)
- Enteral nutrition (B4034-B4152)

### Wheelchair Coding
- Manual wheelchair: K0001-K0108 (complexity levels)
- Power wheelchair: K0800-K0899
- Scooter: K0010-K0014
- Documentation must support medical necessity

## Drug Coding (J Codes)

### How to Code Drug Administration
1. Identify the drug and route
2. Find the J code for the drug
3. Determine number of units (each J code = specific drug amount)
4. Add appropriate administration code (CPT 96360-96379)

### Common J Codes
| Drug | Code |
|------|------|
| Insulin | J1817 |
| Epinephrine 0.1 mg | J0171 |
| Methotrexate 5 mg | J9250 |
| Rituximab 100 mg | J9312 |

## Home Health HCPCS

### Therapy and Home Health
- G0151-G0162: Therapy services in home health
- G0179: MD recertification for home health plan of care
- G0180: MD certification for home health (initial)

### Home Health Coverage Requirements
- Homebound status
- Need for skilled services (nursing, therapy)
- Services must be reasonable and necessary

## Ambulance Coding (A Codes)

### Origin-Destination Modifiers
First character = origin, second = destination:
- D: Diagnostic/therapeutic facility
- E: Residential, domiciliary facility
- G: Hospital
- H: Hospital-based dialysis
- I: Site of accident
- J: Freestanding dialysis
- N: Skilled nursing facility
- P: Physician's office
- R: Residence
- S: Scene of accident
- X: Intermediate stop

Example: -RH = Residence to Hospital
