## DME Coverage Checklist
☑ Medically necessary (physician prescription)
☑ Used in the home
☑ Serves a medical purpose
☑ Durable (withstands repeated use)
☑ CMN on file (if required)

## CMN Required Items
- Oxygen equipment
- Power wheelchairs
- Hospital beds
- CPAP/BiPAP
- Enteral nutrition
- Pneumatic compression devices

## Drug Unit Calculation
```
Units billed = Total ordered dose ÷ Dose per J code unit
Example: J1800 (Paricalcitol 1 mcg)
If 5 mcg ordered: bill 5 units of J1800
```
