# GATE ECE Fundamentals

## About GATE ECE
GATE Electronics and Communication Engineering. Tests core ECE subjects. Used for M.Tech admissions, ISRO, DRDO, PSU jobs.

## Exam Pattern
- 65 questions, 100 marks, 3 hours
- General Aptitude: 15 marks (verbal, numerical, analytical)
- Technical: 85 marks

## Core Subjects

### Engineering Mathematics
- Linear Algebra: matrices, eigenvalues, determinants
- Calculus: limits, differentiation, integration
- Differential Equations: first order, second order
- Complex Analysis: Cauchy's theorem, residues
- Probability: distributions, random variables
- Transform Theory: Laplace, Fourier, Z-transform

### Network Theory
- KVL, KCL, mesh and nodal analysis
- Thevenin, Norton, Superposition theorems
- Transient analysis: RC, RL, RLC circuits
- AC analysis: phasors, impedance, power factor
- Two-port networks: Z, Y, H, ABCD parameters
- Graph theory of circuits

### Electronic Devices
- Semiconductor physics: carriers, drift, diffusion
- PN junction diode: characteristics, models
- BJT: operation, biasing, small-signal models
- MOSFET: NMOS, PMOS, characteristics, models
- Optoelectronics: LED, photodiode, solar cell

### Analog Circuits
- Diode circuits: rectifiers, clippers, clampers
- Transistor amplifiers: CE, CB, CC configurations
- Op-amp: ideal vs real, inverting, non-inverting, integrator, differentiator
- Feedback amplifiers: stability, Bode plots
- Oscillators: Barkhausen criterion, types
- Power amplifiers: classes A, B, AB, C

### Digital Circuits
- Boolean algebra and minimization (K-map, Quine-McCluskey)
- Combinational: adders, multiplexers, decoders, comparators
- Sequential: flip-flops, registers, counters
- ADC, DAC: types and characteristics
- Memory: RAM, ROM, flash

### Signals and Systems
- Classification of signals
- LTI systems: impulse response, convolution
- Fourier series and transform
- Laplace transform and applications
- Z-transform for discrete systems
- Sampling theorem: Nyquist rate

### Control Systems
- Transfer functions, block diagrams
- Signal flow graphs, Mason's formula
- Time response: step response, rise time, overshoot
- Stability: Routh-Hurwitz, Nyquist criterion, Bode plot
- Root locus method
- PID controllers

### Communications
- Amplitude modulation (AM, DSB, SSB)
- Frequency modulation (FM, PM)
- Analog demodulation
- Digital modulation: ASK, FSK, PSK, QAM
- Noise in communications
- Information theory: entropy, channel capacity, Shannon's theorem

### Electromagnetics
- Vector calculus: gradient, divergence, curl
- Maxwell's equations
- Plane waves: propagation, polarization, reflection
- Transmission lines: characteristic impedance, Smith chart
- Waveguides: modes, cutoff frequency
- Antennas: radiation pattern, gain, impedance

### Microprocessors
- 8085/8086 architecture
- Instruction set and addressing modes
- Memory interfacing, I/O
- Interrupts
