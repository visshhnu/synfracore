# Banking Exams Cheatsheet

## Key Formulas — Quantitative Aptitude

### Percentage & Profit-Loss
```
Profit % = (Profit / CP) × 100
Loss % = (Loss / CP) × 100
SP = CP × (100 + P%) / 100
CP = SP × 100 / (100 + P%)
Discount % = Discount / MP × 100
```

### Simple & Compound Interest
```
SI = P × R × T / 100
CI = P × (1 + R/100)^T - P
CI (half-yearly) = P × (1 + R/200)^(2T) - P
Effective Rate for CI = [(1 + R/100)^T - 1] × 100
```

### Time & Work
```
If A completes work in n days → A's 1-day work = 1/n
Combined: 1/A + 1/B = 1/Total
If A:B efficiency ratio = m:n → time ratio = n:m
```

### Speed, Distance, Time
```
Distance = Speed × Time
Relative speed (same dir) = |S1 - S2|
Relative speed (opp dir) = S1 + S2
Time to cross = (L1 + L2) / Relative Speed
```

### Mixtures & Alligation
```
Alligation Rule: (Cheaper - Mean) : (Mean - Dearer)
= quantity of dearer : quantity of cheaper
```

## Reasoning Quick Reference

### Blood Relations
```
Father's brother = Uncle
Mother's sister = Aunt
Son's wife = Daughter-in-law
Daughter's son = Grandson
Brother's son = Nephew
Sister's daughter = Niece
```

### Direction Sense
```
Face North:  Left=West, Right=East, Back=South
Face South:  Left=East, Right=West, Back=North
Sunrise in East, Sunset in West
Shadow: Morning=West, Evening=East, Noon=None
```

### Coding-Decoding Tricks
- Shift cipher: A=1, B=2... or +3/-3 positions
- Reverse alphabet: A=Z, B=Y, C=X...
- Number coding: Count letters (CAT = 3+1+20 = 24)
