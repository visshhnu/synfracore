# Banking Exams — Intermediate

## Advanced Quantitative Aptitude

### Data Interpretation Strategies

**Table DI Approach:**
- Read all column headers before calculation
- Calculate totals first if needed
- Use approximation for percentages (saves time)
- Watch for units: thousands vs lakhs

```
Example Table:
Branches | 2022 | 2023 | Growth %
Delhi    | 2400 | 2760 | ?
Mumbai   | 3100 | 3565 | ?
Chennai  | 1800 | 1980 | ?

Growth % = (New - Old) / Old × 100
Delhi: (2760-2400)/2400 × 100 = 15%
```

**Caselet DI:** Read carefully, extract all values before solving.

### Number Series Patterns
- Arithmetic: difference constant (2,5,8,11...)
- Geometric: ratio constant (2,6,18,54...)
- Square/Cube: (1,4,9,16...) or (1,8,27,64...)
- Fibonacci: sum of previous two (1,1,2,3,5,8...)
- Difference of differences (second-order patterns)

### Probability for Banking
```
P(A) = Favorable outcomes / Total outcomes
P(A∪B) = P(A) + P(B) - P(A∩B)
P(A∩B) = P(A) × P(B) [for independent events]

Permutation: nPr = n!/(n-r)!   [order matters]
Combination: nCr = n!/r!(n-r)! [order doesn't matter]
```

## Advanced Reasoning

### Seating Arrangement — Quick Approach
1. Draw the shape (circle/line) first
2. Fix the anchors (definite positions)
3. Apply conditions one by one
4. Use elimination for ambiguous positions

### Puzzle Solving Framework
1. Read all conditions before drawing grid
2. Mark definite placements first
3. Use "not" markings to eliminate
4. Work through conditional clues last

## Banking Awareness — Key Concepts

### RBI Functions
- Monetary policy (repo rate, reverse repo, CRR, SLR)
- Currency issue and management
- Regulation of banks and NBFCs
- Foreign exchange management
- Payment system oversight

### Key Rates (Always Updated)
| Rate | Purpose |
|------|---------|
| Repo Rate | Rate at which RBI lends to banks |
| Reverse Repo | Rate at which banks park funds with RBI |
| CRR | Cash Reserve Ratio — % of NDTL kept with RBI |
| SLR | Statutory Liquidity Ratio — % in govt securities |
| Bank Rate | Long-term lending rate |
| MSF | Marginal Standing Facility — emergency borrowing |

### Priority Sector Lending
Banks must lend 40% of ANBC to priority sectors:
- Agriculture (18%)
- Micro, Small & Medium Enterprises
- Education loans
- Housing loans (up to limits)
- Renewable energy

## English — Reading Comprehension Strategy
1. Skim passage first (30 seconds) for main idea
2. Read questions before detailed reading
3. Go back to passage for answers (don't rely on memory)
4. Elimination works well for vocab questions
5. Para-based questions: read surrounding paras too

## Mock Test Analysis Framework
After every mock test:
- Note time taken per section
- Identify which question types took longest
- Mark questions answered wrong vs. skipped
- Revisit wrong answers without peeking at solutions
- Create error log by topic
