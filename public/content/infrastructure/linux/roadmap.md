# Linux — Learning Roadmap

## Time to Job-Ready: 8 weeks

**Goal:** Can administer Linux servers, write automation scripts, diagnose performance issues

---

## Phase 1: Week 1-2: Shell Fundamentals

- Navigate filesystem (ls, cd, pwd)
- File operations (cp, mv, rm, chmod)
- Text processing (grep, sed, awk, cat)
- Redirects and pipes (|, >, >>, 2>&1)

## Phase 2: Week 3-4: System Administration

- Process management (ps, top, kill, systemctl)
- Package management (apt/yum/dnf)
- User and group management
- Cron jobs and scheduling

## Phase 3: Week 5-6: Networking & Security

- Networking tools (ss, ip, curl, netstat)
- SSH configuration and key management
- Firewall (ufw/iptables) basics
- File permissions and sudo deep dive

## Phase 4: Week 7-8: Scripting & Performance

- Bash scripting (variables, loops, functions)
- System monitoring (vmstat, iostat, sar)
- Log analysis and rotation
- Disk management (lvm, df, du, fdisk)

## Resources

- Linux Command Line by William Shotts (free online)
- OverTheWire: Bandit (wargame)
- TLDR.sh for quick command reference
- man pages and --help flags

## Certifications to Target

- RHCSA (EX200) — Red Hat Certified System Administrator
- Linux Foundation Certified System Administrator (LFCS)
