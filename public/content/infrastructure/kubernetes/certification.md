# Kubernetes — Certification Guide

## CKA — Certified Kubernetes Administrator

| Detail | Info |
|---|---|
| **Provider** | CNCF |
| **Exam Code** | CKA |
| **Cost** | $395 |
| **Validity** | 3 years |
| **Format** | 2 hours, performance-based (hands-on terminal) |

### Domain Breakdown

- **25%** — Cluster Architecture, Installation & Configuration
- **15%** — Workloads & Scheduling
- **20%** — Services & Networking
- **10%** — Storage
- **30%** — Troubleshooting

## CKAD — Certified Kubernetes Application Developer

| Detail | Info |
|---|---|
| **Provider** | CNCF |
| **Exam Code** | CKAD |
| **Cost** | $395 |
| **Validity** | 3 years |
| **Format** | 2 hours, performance-based |

### Domain Breakdown

- **20%** — Application Design & Build
- **20%** — Application Deployment
- **25%** — Application Observability & Maintenance
- **25%** — Application Environment, Configuration & Security
- **10%** — Services & Networking

## Exam Tips

- Book exam + free retake bundle
- Use killer.sh (2 free mock exams with purchase)
- Practice fast kubectl: aliases, imperative commands
- --dry-run=client -o yaml is your best friend
- Know the docs structure — you can access kubernetes.io during exam
