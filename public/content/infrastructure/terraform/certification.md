# Terraform — Certification Guide

## HashiCorp Terraform Associate 003

| Detail | Info |
|---|---|
| **Provider** | HashiCorp |
| **Exam Code** | TA-003 |
| **Cost** | $70 |
| **Validity** | 2 years |
| **Format** | 60 questions, 1 hour, MCQ + true/false |

### Domain Breakdown

- **11%** — Understand IaC concepts
- **11%** — Understand Terraform purpose vs other IaC
- **9%** — Understand Terraform basics
- **22%** — Navigate Terraform workflow
- **11%** — Implement and maintain state
- **6%** — Read, generate, and modify configuration
- **9%** — Understand Terraform Cloud capabilities

## Exam Tips

- Complete HashiCorp Learn tutorials (free, official)
- Understand state locking, workspaces, remote backends
- Know difference between plan/apply/destroy/import
- Understand count vs for_each vs dynamic blocks
- Study Terraform Cloud: remote runs, sentinel policies
