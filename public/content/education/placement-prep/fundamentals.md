# Placement Preparation — Fundamentals

## The Placement Process

```
Typical Tech Company Process:
  Round 1: Online Test (MCQ + Coding, 60-90 min)
    - Aptitude (Quant, Logical, Verbal)
    - Technical MCQ (OS, DBMS, CN, C/Java basics)
    - 2 Coding problems (Easy-Medium level)
  
  Round 2: Technical Interview 1 (45-60 min)
    - DSA: 2-3 coding problems on whiteboard/screen
    - CS fundamentals: OS, DBMS, CN concepts
  
  Round 3: Technical Interview 2 (45-60 min)
    - System design (experienced candidates)
    - Project discussion
    - Advanced DS&A
  
  Round 4: HR Interview (30 min)
    - Behavioral questions (STAR method)
    - Salary negotiation
    - Company culture fit
```

## Coding Interview Strategy

```python
# UMPIRE Method:
# Understand: Repeat the problem, ask clarifying questions
# Match: Identify the pattern (two pointers, BFS, DP, etc.)
# Plan: Write pseudocode before coding
# Implement: Clean code with meaningful variable names
# Review: Test with examples (empty input, single element, large input)
# Evaluate: Time and space complexity

# Common patterns to master:
# Arrays/Strings: Two pointers, Sliding window, Prefix sum
# Sorting: QuickSort, MergeSort, Counting sort
# Searching: Binary search (and variants)
# Trees: BFS, DFS, LCA, BST operations
# Graphs: BFS shortest path, DFS cycle detection, Topological sort
# Dynamic Programming: 1D, 2D, Knapsack
# Heap: K largest/smallest, Median

# Must-know LeetCode problems:
# Two Sum, Valid Parentheses, Merge Intervals
# Maximum Subarray (Kadane's), Best Time to Buy Stock
# Reverse Linked List, Detect Cycle
# Binary Tree Level Order, LCA
# Climbing Stairs, Coin Change, Longest Common Subsequence
```

## Aptitude Quick Formulas

```
Number Series:
  AP (Arithmetic Progression): a, a+d, a+2d... Sum = n/2(2a+(n-1)d)
  GP (Geometric): a, ar, ar²... Sum = a(r^n - 1)/(r-1)

Percentages:
  X% of Y = XY/100
  % increase = (new-old)/old × 100
  Successive discount: 20% + 10% = 28% (not 30%)

Time & Work:
  A completes in m days, B in n days: Together = mn/(m+n) days

Time, Speed, Distance:
  D = S × T (Distance = Speed × Time)
  Average speed = Total distance / Total time (NOT avg of speeds)

Probability:
  P(A) = favorable outcomes / total outcomes
  P(A and B) = P(A) × P(B) [if independent]
  P(A or B) = P(A) + P(B) - P(A and B)

Permutation & Combination:
  nPr = n!/(n-r)!    (arrangement, order matters)
  nCr = n!/(r!(n-r)!) (selection, order doesn't matter)
```
