# Vocabulary and Grammar for SSC

## Why This Chapter Matters
English carries 25-50 marks in SSC CGL (Tier 1: 25 marks, Tier 2 English paper: 200 marks). Vocabulary (synonyms, antonyms, idioms, one-word) and Grammar (error detection, sentence improvement) are the core components.

## Grammar

### 5. Common Error Types (SSC Pattern)

**Subject-Verb Agreement errors:**
"Each of the students have submitted" → has (Each = singular)
"The committee have decided" → has (committee = one body)
"A number of students are" → ✓ (a number of = plural)
"The number of students is" → ✓ (the number of = singular)

**Tense errors:**
"Yesterday I have gone to market" → went (yesterday = simple past)
"She has been here since two days" → for (duration = for, point = since)

**Preposition errors:**
"Congratulate on" ✓ | "Die of disease" ✓ | "Die from wound" ✓
"Suffer from" ✓ | "Superior to" ✓ (not than) | "Married to" ✓

**Article errors:**
"He is an MBA" ✓ (vowel sound) | "A honest man" ✗ → an (h silent)
"She is an European" ✗ → a (E sounds like 'yoo') 

### 6. Sentence Improvement
Read for: wrong tense, wrong preposition, wrong article, redundancy, wrong comparison.
"More superior" → superior (comparative already in word)
"Return back" → return (back is redundant)
"Reason is because" → reason is that

## PYQs (SSC)

**SSC CGL 2023 (Synonym):** FORTUITOUS
(A) Strong (B) Fortunate (C) Accidental ✓ (D) Deliberate
[Fortuitous = happening by chance, accidental]

**SSC CHSL 2022 (Idiom):** "She was over the moon when she got the job."
(A) Very sad (B) Very happy ✓ (C) On the moon (D) Confused

**SSC CGL 2022 (Error):** "He don't know the answer to this question."
Error: "don't" → "doesn't" (He = third person singular → doesn't)

## Revision Notes
```
VOCABULARY STRATEGY:
Learn words in groups (synonyms of "shrewd": astute, sagacious, perspicacious, cunning)
Learn idioms with example sentences, not just meanings
One-word substitutions: focus on -cide (killings), -phobia (fears), -ology (studies)

GRAMMAR: Most common SSC errors:
1. Subject-verb agreement (Each/Every/Neither/Either → singular)
2. Preposition (die OF, congratulate ON, superior TO, prefer TO)
3. Tense (yesterday → simple past; since point, for duration)
4. Article (an + vowel SOUND; a + consonant SOUND)
5. Redundancy (return back, revert back, end result)
```
