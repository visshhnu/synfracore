## Core Concepts

### 1. Standard Form of a Quadratic Equation

$$ax^2 + bx + c = 0 \quad \text{where } a \neq 0$$

- a, b, c are real numbers (a ‚Č† 0 ‚ÄĒ if a = 0, it becomes linear)
- a = coefficient of x¬≤ (called **leading coefficient**)
- b = coefficient of x
- c = constant term

**Examples:**
- 2x¬≤ ‚ąí 5x + 3 = 0 ‚úď (a=2, b=‚ąí5, c=3)
- x¬≤ = 3x ‚Üí x¬≤ ‚ąí 3x = 0 ‚úď (a=1, b=‚ąí3, c=0)
- (x+1)(x‚ąí2) = 0 ‚Üí x¬≤ ‚ąí x ‚ąí 2 = 0 ‚úď

**NOT quadratic:** x¬≥ + x = 0 (degree 3), 5x + 3 = 0 (degree 1)

### 2. Method 1 ‚ÄĒ Factorisation (Splitting the Middle Term)

**Steps:**
1. Write equation in standard form ax¬≤ + bx + c = 0
2. Find two numbers whose **sum = b** and **product = ac**
3. Split the middle term using these two numbers
4. Group and factorise
5. Apply Zero Product Property: if pq = 0, then p = 0 or q = 0

**Example:** Solve 6x¬≤ ‚ąí x ‚ąí 2 = 0

Step 1: a=6, b=‚ąí1, c=‚ąí2 ‚Üí ac = 6√ó(‚ąí2) = ‚ąí12
Step 2: Find two numbers with sum = ‚ąí1 and product = ‚ąí12 ‚Üí 3 and ‚ąí4 (3 + (‚ąí4) = ‚ąí1, 3√ó(‚ąí4) = ‚ąí12)
Step 3: 6x¬≤ + 3x ‚ąí 4x ‚ąí 2 = 0
Step 4: 3x(2x + 1) ‚ąí 2(2x + 1) = 0 ‚Üí (3x ‚ąí 2)(2x + 1) = 0
Step 5: x = 2/3 or x = ‚ąí1/2

---

### 3. Method 2 ‚ÄĒ Completing the Square

**Steps:**
1. Make coefficient of x¬≤ equal to 1 (divide throughout by a)
2. Move constant to right side
3. Add (b/2a)¬≤ to both sides
4. Write left side as a perfect square
5. Take square root of both sides

**Example:** Solve 2x¬≤ + x ‚ąí 4 = 0

```
Step 1: x¬≤ + x/2 ‚ąí 2 = 0           (divide by 2)
Step 2: x¬≤ + x/2 = 2
Step 3: x¬≤ + x/2 + (1/4)¬≤ = 2 + 1/16    (add (b/2)¬≤ = (1/4)¬≤ = 1/16)
Step 4: (x + 1/4)¬≤ = 32/16 + 1/16 = 33/16
Step 5: x + 1/4 = ¬Ī‚ąö33/4
        x = (‚ąí1 ¬Ī ‚ąö33)/4
```

### 4. Method 3 ‚ÄĒ Quadratic Formula (Most Important)

For ax¬≤ + bx + c = 0:

$$\boxed{x = \frac{-b \pm \sqrt{b^2 - 4ac}}{2a}}$$

This formula ALWAYS works. Learn it by heart.

**Example:** Solve x¬≤ ‚ąí 7x ‚ąí 8 = 0

a=1, b=‚ąí7, c=‚ąí8

$$x = \frac{7 \pm \sqrt{49 + 32}}{2} = \frac{7 \pm \sqrt{81}}{2} = \frac{7 \pm 9}{2}$$

x = (7+9)/2 = 8 OR x = (7‚ąí9)/2 = ‚ąí1

### 5. Discriminant and Nature of Roots (Most Tested!)

The expression **D = b¬≤ ‚ąí 4ac** is called the **discriminant**.

| Discriminant (D) | Nature of Roots | How many real roots? |
|---|---|---|
| D > 0 | Two distinct real roots | 2 |
| D = 0 | Two equal real roots (repeated root) | 1 (counted twice) |
| D < 0 | No real roots (complex roots) | 0 |

**Example 1:** Find the nature of roots of 2x¬≤ ‚ąí 4x + 3 = 0
D = (‚ąí4)¬≤ ‚ąí 4(2)(3) = 16 ‚ąí 24 = **‚ąí8 < 0**
‚Üí No real roots exist.

**Example 2:** Find k if kx¬≤ + 4x + 1 = 0 has equal roots
For equal roots: D = 0 ‚Üí 16 ‚ąí 4k = 0 ‚Üí **k = 4**

### 6. Word Problems ‚ÄĒ Setting Up the Equation

**Step 1:** Read carefully, identify the unknown ‚ÄĒ let it be x
**Step 2:** Form expressions for other quantities in terms of x
**Step 3:** Use the given condition to form an equation
**Step 4:** Solve the equation
**Step 5:** Check whether both/one answer is valid in context (lengths/ages can't be negative)

## Solved Examples

### Example 1 ‚ÄĒ Factorisation
**Q:** Find roots of x¬≤ ‚ąí 3x ‚ąí 10 = 0

Product = ‚ąí10, Sum = ‚ąí3 ‚Üí numbers: ‚ąí5 and 2
x¬≤ ‚ąí 5x + 2x ‚ąí 10 = 0
x(x‚ąí5) + 2(x‚ąí5) = 0
(x+2)(x‚ąí5) = 0
**x = ‚ąí2 or x = 5**

---

### Example 2 ‚ÄĒ Quadratic Formula
**Q:** Solve 2x¬≤ + 7x + 3 = 0

D = 49 ‚ąí 24 = 25
x = (‚ąí7 ¬Ī 5)/4
x = ‚ąí2/4 = **‚ąí1/2** OR x = ‚ąí12/4 = **‚ąí3**

---

### Example 3 ‚ÄĒ Word Problem (Classic)
**Q:** The product of two consecutive positive integers is 306. Find the integers.

Let smaller integer = x ‚Üí larger = x + 1
x(x + 1) = 306
x¬≤ + x ‚ąí 306 = 0
D = 1 + 1224 = 1225 = 35¬≤
x = (‚ąí1 ¬Ī 35)/2 ‚Üí x = 17 (positive) or x = ‚ąí18 (rejected, not positive)

**Answer: The integers are 17 and 18.** Check: 17 √ó 18 = 306 ‚úď

---

### Example 4 ‚ÄĒ Speed/Distance Problem
**Q:** A train travels 360 km at a uniform speed. If speed had been 5 km/h more, it would have taken 1 hour less. Find the speed.

Let speed = x km/h
Time at speed x = 360/x hours
Time at speed (x+5) = 360/(x+5) hours
Difference = 1: 360/x ‚ąí 360/(x+5) = 1

360(x+5) ‚ąí 360x = x(x+5)
1800 = x¬≤ + 5x
x¬≤ + 5x ‚ąí 1800 = 0
(x + 45)(x ‚ąí 40) = 0
x = 40 or x = ‚ąí45 (rejected)

**Speed = 40 km/h**

## Revision Notes

```
Quadratic Formula:  x = (‚ąíb ¬Ī ‚ąöD) / 2a
Discriminant:       D = b¬≤ ‚ąí 4ac

Nature of Roots:
  D > 0 ‚Üí Two distinct real roots
  D = 0 ‚Üí Two equal roots (each = ‚ąíb/2a)
  D < 0 ‚Üí No real roots

Factorisation shortcut:
  Find p, q such that p + q = b and pq = ac
  Then ax¬≤ + bx + c = ax¬≤ + px + qx + c

Word problem strategy:
  1. Let unknown = x
  2. Express everything in terms of x
  3. Form equation from given condition
  4. Solve and reject invalid answers
```

**Common Mistakes:**
‚ĚĆ Forgetting that negative roots may need to be rejected in real-life problems (age, length, speed are always positive)
‚ĚĆ In formula, forgetting the ¬Ī (both + and ‚ąí)
‚ĚĆ D = b¬≤ ‚ąí 4ac, NOT b¬≤ + 4ac ‚ÄĒ the sign is MINUS
