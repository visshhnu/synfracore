# C Programming

C is the mother of modern programming languages. Created in 1972 at Bell Labs by Dennis Ritchie, C powers operating systems (Linux, Windows kernel), embedded systems, databases (MySQL, PostgreSQL, SQLite), compilers, and virtually every piece of critical infrastructure. Learning C gives you deep understanding of how computers actually work.

## Why C Still Matters

- **Foundation** — C, C++, Java, Python all have C roots. Master C = understand everything
- **GATE/Placement** — Required for systems programming questions
- **Embedded systems** — IoT, microcontrollers, firmware all use C
- **Performance** — Fastest language after assembly; used where speed is critical
- **OS internals** — Linux kernel is C; understanding C = understanding OS

## C vs Higher-Level Languages

```
C:       You control everything — memory, pointers, hardware
Python:  Memory auto-managed, slower, easier syntax
Java:    Memory auto-managed (GC), platform-independent, verbose
Assembly: Even lower level, hardware-specific, extremely difficult

Rule: Higher level = easier to write, slower/less control
      Lower level = harder to write, faster/more control
```
