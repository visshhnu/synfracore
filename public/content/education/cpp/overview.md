# C++ Programming

C++ is a powerful systems and application programming language that adds object-oriented programming, templates, and the Standard Template Library (STL) to C. It's the language of choice for competitive programming, game development, system software, and high-performance applications.

## Why Learn C++

- **Competitive programming** — fastest language; almost all top competitive programmers use C++
- **Placement/GATE** — many interviews accept C++; STL data structures are essential
- **Game development** — Unreal Engine, most AAA games written in C++
- **High-performance systems** — financial systems, browsers (Chrome/V8), databases
- **C++ + STL** — once mastered, lets you code DSA problems 2-3x faster than Java/Python

## C++ vs C vs Java

```
C++:    Object-oriented + systems control, manual memory OR RAII
C:      Pure systems, maximum control, no OOP
Java:   OOP with garbage collection, slower, verbose
Python: Slowest, easiest syntax, best for AI/scripts
```
