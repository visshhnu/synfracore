# Puzzles and Seating Arrangement

## Why This Chapter Matters
Puzzles and Seating Arrangement is the HIGHEST WEIGHTAGE topic in Banking Reasoning — 10-15 marks per paper in SBI PO, IBPS PO, and RRB. Complex puzzles appear in every single exam.

## Types of Puzzles

### 1. Linear Arrangement
People sitting in a row (single or double row).
Key clues: "to the left/right of", "immediate neighbour", "not adjacent", "at one of the ends"

**Double row arrangement:**
Two rows facing each other. "Opposite to" = across the table.
Person in row 1 faces person in row 2 directly across.

**Step-by-step approach:**
1. Read all clues quickly, note constraints.
2. Start with the most definitive clue (fixes a person's position absolutely).
3. Build outward using relative clues.
4. Cross-check all clues after solving.

**Example:**
5 people A,B,C,D,E sitting in a row. B is 3rd from left. A is 2nd from right. C is to the immediate right of B. D is not adjacent to A. 
Positions: _  _  B  _  _. C is to right of B → position 4. A is 2nd from right → position 4. Contradiction! Re-read...
A is 2nd from right → position 4. B is 3rd from left → position 3. C to right of B → position 4. C=A? If A and C are different, conflict → check if "2nd from right" = position 4 in 5 seats. Yes. Means clue says A is at position 4, C is also at 4 → re-examine. C is to IMMEDIATE right = position 4, A is also at 4 → only works if A=position 5 (rightmost) and "2nd from right" means position 4 = C. Then: _ _ B C A. D not adjacent to A → D not at position 4 (C is there). D at 1 or 2. E gets remaining.

### 2. Circular Arrangement
People sitting around a circular table.

**Two types:**
Facing inward (toward center): RIGHT hand side = clockwise when looking from above
Facing outward (away from center): RIGHT hand side = anti-clockwise when looking from above

**Key:** Always specify facing direction! "Immediate right" depends on facing.

**Approach:**
1. Fix one person (any) at top of circle as reference.
2. Place others using clues relative to first person.
3. Check ALL clues before finalising.

### 3. Floor/Building Puzzle
People on different floors (1 = ground, N = top) or (1 = top, N = bottom).
Clues: "above", "below", "adjacent floor", "between floors X and Y"

**Approach:**
1. List all floors.
2. Start with direct clues (person on specific floor).
3. Use relative clues to fill in.

### 4. Box/Month/Day Puzzle
Boxes stacked (top to bottom), or months in a year, or days of the week.
Similar approach — list all positions, fill with definitive clues first.

### 5. Complex/Multi-variable Puzzle
Combines 2-3 parameters: person + city + profession, or person + colour + floor + day.
Use a TABLE to track. Columns = parameters. Rows = people.
Fill confirmed cells with ✓, eliminate with ✗.

## Strategy for Solving

**Rule 1:** Never assume. Use only given clues.
**Rule 2:** Start with the most definitive clue.
**Rule 3:** For complex puzzles, use elimination — mark what's NOT possible.
**Rule 4:** When stuck, try assuming one option and see if contradiction arises.
**Rule 5:** Verify ALL clues before answering questions.

## Time Management
In exam: 5 questions from 1 puzzle set. Spend max 7-8 minutes on one set.
If stuck after 5 minutes: move on, come back later.

## Previous Year Pattern
**SBI PO 2023:** Circular arrangement of 8 people alternating men and women. Complex clues with likes/dislikes.
**IBPS PO 2022:** Double row of 6 people each. One row facing north, other facing south.
**IBPS Clerk 2022:** Building with 6 floors. 6 people + profession. Multi-variable.

## Revision Notes
```
LINEAR: Row position 1 = leftmost. "Immediate right of P" = next to P's right.
CIRCULAR INWARD: A's right = clockwise neighbour.
CIRCULAR OUTWARD: A's right = anti-clockwise neighbour.

APPROACH (ANY PUZZLE):
1. List all positions/slots
2. Start with most definitive clue
3. Work outward using relative clues
4. Verify ALL clues at end

COMMON TRAPS:
"Second from left" ≠ "Third from right" (unless n=5+: always calculate)
"Facing north" in circular ≠ same as "facing inward"
"Not adjacent" means neither left nor right
"Between A and B" = someone sits between A and B (at least 1 gap)
```
