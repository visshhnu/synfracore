# JTO Exam Quick Notes

## Formula Sheet

### Communication
- AM bandwidth: BW = 2 × fm
- FM bandwidth (Carson's rule): BW = 2(Δf + fm)
- Shannon capacity: C = B log₂(1 + S/N)
- Nyquist sampling rate: fs ≥ 2 × fmax
- PCM bits per sample: n = log₂(L) where L = quantization levels
- SNR for PCM: SNR = 6.02n + 1.76 dB

### Antennas
- Free space path loss: FSPL = (4πd/λ)²
- Half-wave dipole length: L = λ/2
- Effective aperture: Ae = Gλ²/4π

### Transmission Lines
- Phase velocity: vp = 1/√(LC) = c/√(εr)
- Wavelength: λ = vp/f
- Input impedance of λ/4 transformer: Zin = Z₀²/ZL
