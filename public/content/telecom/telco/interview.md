# Telecom Industry Interview Q&A

**Q: What is OSS vs BSS?**
OSS (Operations Support System): manage network — fault management, performance, configuration, inventory. BSS (Business Support System): manage business — billing, customer management, order management, product catalog.

**Q: What is the difference between 4G LTE and 5G?**
4G: OFDMA air interface, EPC core, peak 100Mbps, ~10ms latency. 5G: OFDM + mmWave, 5GC service-based architecture, peak 20Gbps, <1ms URLLC latency, network slicing, massive MIMO.

**Q: What is RAN?**
Radio Access Network: the part of mobile network between device and core. Consists of: antennas, base stations (eNodeB for 4G, gNodeB for 5G). RAN connects UE to EPC/5GC via air interface.
