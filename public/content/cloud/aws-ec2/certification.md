# EC2 — Certification Guide

## AWS Cloud Practitioner (CLF-C02)

| Detail | Info |
|---|---|
| **Provider** | AWS |
| **Exam Code** | CLF-C02 |
| **Cost** | $100 |
| **Validity** | 3 years |
| **Format** | 65 questions, 90 min, MCQ |

### Domain Breakdown

- **24%** — Cloud Concepts
- **30%** — Security & Compliance
- **17%** — Cloud Technology & Services
- **29%** — Billing, Pricing & Support

## AWS Solutions Architect Associate (SAA-C03)

| Detail | Info |
|---|---|
| **Provider** | AWS |
| **Exam Code** | SAA-C03 |
| **Cost** | $150 |
| **Validity** | 3 years |
| **Format** | 65 questions, 130 min, MCQ + multi-select |

### Domain Breakdown

- **30%** — Design Resilient Architectures
- **28%** — Design High-Performing Architectures
- **24%** — Design Secure Architectures
- **18%** — Design Cost-Optimized Architectures

## Exam Tips

- Stephane Maarek (Udemy) courses are the gold standard
- Practice with AWS Free Tier — hands-on beats videos
- Focus on which service to use WHEN, not just what it does
- Well-Architected Framework is heavily tested: 6 pillars
- Practice exams: Tutorial Dojo (Jon Bonso) is highly recommended
