# CPT Coding — Fundamentals

## CPT Code Structure

```
CPT (Current Procedural Terminology) — AMA-maintained code set
Used to report: medical, surgical, diagnostic, and therapeutic services

Code ranges:
  00100-01999: Anesthesia
  10004-69990: Surgery
  70010-79999: Radiology
  80047-89398: Pathology and Laboratory
  90281-99607: Medicine
  99202-99215: Office/Outpatient E/M visits
  99221-99233: Hospital inpatient E/M visits

Modifiers (2-digit codes added after):
  -25: Significant, separately identifiable E/M same day as procedure
  -51: Multiple procedures
  -59: Distinct procedural service
  -RT/-LT: Right/Left side
  -TC: Technical component only
  -26: Professional component only
```

## E/M Coding (Key for Outpatient)

```
Office/Outpatient Visit codes (2021 guidelines):
  99202: New patient, straightforward MDM, 15-29 min
  99203: New patient, low MDM, 30-44 min
  99204: New patient, moderate MDM, 45-59 min
  99205: New patient, high MDM, 60-74 min
  
  99212: Established, straightforward MDM, 10-19 min
  99213: Established, low MDM, 20-29 min
  99214: Established, moderate MDM, 30-39 min
  99215: Established, high MDM, 40-54 min

MDM = Medical Decision Making
  - Number/complexity of problems addressed
  - Data reviewed (tests, records, independent interpretation)
  - Risk (prescription drugs, procedures, hospitalization risk)
```
